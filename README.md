# Leer JavaScript - Complete Leerwebsite

Een interactieve educatieve website om alles over JavaScript te leren, van basis tot geavanceerd niveau.

## Features

### 📚 Leerinhoud
- **Basis Onderwerpen**: Variabelen, Data Types, Operators, Loops, Conditionals, Functions
- **Geavanceerde Onderwerpen**: Objects & Classes, Arrays & Methods, Async & Promises, DOM Manipulatie, Events, ES6+ Features

### 🎯 Interactieve Elementen
- **Voorbeelden**: Klik op "Bekijk Voorbeeld" om code voorbeelden te zien
- **Code Uitvoeren**: Run code direct in de browser om te zien wat het doet
- **Oefeningen**: 3 praktische oefeningen met directe feedback
- **Playground**: Vrije JavaScript code editor om te experimenteren

### 💡 Functionaliteit
- Smooth scrolling navigatie
- Responsive design voor alle apparaten
- Code syntax highlighting
- Real-time code execution
- Error handling en feedback
- Mobile-friendly hamburger menu

## Gebruik

1. Open `index.html` in je browser
2. Navigeer door de verschillende secties
3. Klik op "Bekijk Voorbeeld" om code voorbeelden te zien
4. Gebruik de "Run Code" knoppen om code uit te voeren
5. Probeer de oefeningen in de "Oefeningen" sectie
6. Experimenteer in de "JavaScript Playground"

## Secties

### 🏠 Home
Hero sectie met introductie en een "Hello World" voorbeeld

### 📖 Basis
6 basis onderwerpen met uitleg en voorbeelden:
- Variabelen (let, const, var)
- Data Types (String, Number, Boolean, etc.)
- Operators (rekenkundig, vergelijking, logisch)
- Loops (for, while, forEach)
- Conditionals (if/else, switch)
- Functions (declarations, arrow functions)

### 🚀 Geavanceerd
6 geavanceerde onderwerpen:
- Objects & Classes
- Arrays & Methods (map, filter, reduce)
- Async & Promises
- DOM Manipulatie
- Events
- ES6+ Features

### 💻 Voorbeelden
Interactieve tab interface met 4 voorbeelden die je direct kunt uitvoeren

### ✏️ Oefeningen
3 praktische oefeningen:
1. Variabelen maken
2. Functions schrijven
3. Loops gebruiken

### 🎮 Playground
Vrije code editor waar je JavaScript code kunt schrijven en uitvoeren

## Bestanden

- `index.html` - Hoofdstructuur van de website
- `styles.css` - Styling met JavaScript thema (geel/zwart)
- `script.js` - Alle interactieve functionaliteit en voorbeelden

## Design

- **Kleuren**: JavaScript geel (#F7DF1E) met donkere achtergrond
- **Code Editor**: Donkere theme met syntax highlighting
- **Responsive**: Werkt perfect op desktop, tablet en mobiel

## Browser Ondersteuning

Werkt in alle moderne browsers (Chrome, Firefox, Safari, Edge).

- Probeer alle oefeningen om je kennis te testen
- Bekijk de voorbeelden en probeer ze aan te passen

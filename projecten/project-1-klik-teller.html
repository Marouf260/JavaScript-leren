Project 1: Klik Teller

Beginner

Maak een knop die een teller verhoogt bij elke klik.

📚 Wat ga je leren?

In dit project leer je de basis van DOM manipulatie en event handling. Je leert hoe je:

  • Elementen selecteert uit de HTML
  • Event listeners toevoegt aan elementen
  • Variabelen gebruikt om state bij te houden
  • De DOM dynamisch update wanneer iets gebeurt

🎯 Belangrijke Concepten

  • getElementById() - Selecteert een element op basis van zijn ID
  • addEventListener() - Voegt een event listener toe aan een element
  • textContent - Eigenschap om tekst in een element te wijzigen
  • Variabelen (let/const) - Om data op te slaan die kan veranderen
  • Events - Gebeurtenissen zoals klikken, typen, etc.

📝 Stap-voor-Stap Uitleg

Stap 1: Variabele maken

We beginnen met een variabele die de huidige teller waarde bijhoudt:

let teller = 0; // Begin bij 0

Uitleg: let maakt een variabele die je later kunt aanpassen. We beginnen bij 0 omdat de teller nog niet is gebruikt.

Stap 2: Elementen selecteren

We moeten de knop en het display element uit de HTML halen:

const knop = document.getElementById('counter-btn');
const display = document.getElementById('counter-display');

Uitleg: document.getElementById() zoekt een element met een specifieke ID. We gebruiken const omdat deze referenties niet veranderen.

Stap 3: Event Listener toevoegen

We luisteren naar klik events op de knop:

knop.addEventListener('click', function() {
    // Code die uitgevoerd wordt bij klik
});

Uitleg: addEventListener voegt een functie toe die wordt uitgevoerd wanneer het 'click' event gebeurt op de knop.

Stap 4: Teller verhogen en updaten

Bij elke klik verhogen we de teller en updaten we de display:

knop.addEventListener('click', function() {
    teller++; // Verhoog teller met 1
    display.textContent = 'Teller: ' + teller; // Update tekst
});

Uitleg: teller++ is hetzelfde als teller = teller + 1. textContent wijzigt de zichtbare tekst in het element.

💻 Complete Code

// Stap 1: Variabele voor teller
let teller = 0;

// Stap 2: Selecteer elementen
const knop = document.getElementById('counter-btn');
const display = document.getElementById('counter-display');

// Stap 3 & 4: Event listener met logica
knop.addEventListener('click', function() {
    teller++; // Verhoog teller
    display.textContent = 'Teller: ' + teller; // Update display
});

✅ Instructies:

  • Maak een variabele om de teller bij te houden (begin bij 0)
  • Selecteer de knop element met getElementById
  • Selecteer het display element
  • Voeg een click event listener toe aan de knop
  • Verhoog de teller bij elke klik met ++
  • Update de tekst met textContent

💡 Tips & Tricks

  • Waarom let en const? Gebruik let voor variabelen die veranderen (zoals teller), en const voor elementen die niet veranderen.
  • Alternatief voor textContent: Je kunt ook innerHTML gebruiken, maar textContent is veiliger tegen XSS aanvallen.
  • Arrow functions: Je kunt ook () => {} gebruiken in plaats van function() {}
  • Debuggen: Voeg console.log(teller) toe om de waarde te zien in de console.

⚠️ Veelvoorkomende Fouten

Fout 1: Vergeten elementen te selecteren

// ❌ FOUT - Element bestaat nog niet
knop.addEventListener('click', function() {
    teller++;
});

// ✅ GOED - Eerst selecteren
const knop = document.getElementById('counter-btn');
knop.addEventListener('click', function() {
    teller++;
});

Fout 2: Verkeerde ID naam

// ❌ FOUT - ID bestaat niet
const knop = document.getElementById('button'); // Moet 'counter-btn' zijn

// ✅ GOED - Correcte ID
const knop = document.getElementById('counter-btn');

Fout 3: Teller niet verhogen

// ❌ FOUT - Teller blijft altijd 0
display.textContent = 'Teller: ' + teller; // Vergeet teller++

// ✅ GOED - Verhoog eerst
teller++;
display.textContent = 'Teller: ' + teller;

🚀 Uitbreidingsmogelijkheden

Probeer deze uitdagingen om je vaardigheden te verbeteren:

  • Reset knop: Voeg een knop toe die de teller terugzet naar 0
  • Verlaag knop: Voeg een knop toe die de teller verlaagt
  • Kleur verandering: Verander de kleur van de teller wanneer deze boven 10 komt
  • Geluid: Speel een geluid af bij elke klik
  • LocalStorage: Sla de teller waarde op zodat deze behouden blijft na refresh

Voorbeeld: Reset Functionaliteit

// Voeg een reset knop toe in HTML:
// <button id="reset-btn">Reset</button>

const resetBtn = document.getElementById('reset-btn');
resetBtn.addEventListener('click', function() {
    teller = 0;
    display.textContent = 'Teller: 0';
});

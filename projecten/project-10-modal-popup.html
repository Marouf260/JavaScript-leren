Project 10: Modal Popup

Intermediate

Maak een modal popup die opent en sluit.

📚 Wat ga je leren?

In dit project leer je werken met overlays, z-index en event bubbling. Je leert:

  • Fixed positioning gebruiken voor overlays
  • z-index gebruiken voor layering
  • Event target gebruiken om klik locatie te detecteren
  • Body overflow controleren om scroll te voorkomen
  • Herbruikbare functies maken

🎯 Belangrijke Concepten

  • position: fixed - Positioneert element relatief aan viewport
  • z-index - CSS eigenschap om elementen boven elkaar te leggen
  • event.target - Het element waarop geklikt is
  • body.style.overflow - Controleert scrollbaarheid van pagina
  • Event bubbling - Events gaan omhoog door DOM tree

📝 Stap-voor-Stap Uitleg

Stap 1: Elementen selecteren

Selecteer alle benodigde elementen:

const openBtn = document.getElementById('open-modal');
const overlay = document.getElementById('modal-overlay');
const closeBtn = document.getElementById('close-modal');

Uitleg: We hebben drie elementen nodig: knop om te openen, overlay voor achtergrond, en knop om te sluiten.

Stap 2: Open modal functie

Toon modal en voorkom body scroll:

openBtn.addEventListener('click', function() {
    overlay.style.display = 'flex';
    document.body.style.overflow = 'hidden';
});

Uitleg: display: flex toont overlay en centreert content. overflow: hidden voorkomt scroll op achtergrond.

Stap 3: Close modal functie

Maak herbruikbare functie om modal te sluiten:

function closeModal() {
    overlay.style.display = 'none';
    document.body.style.overflow = 'auto';
}

Uitleg: Door een functie te maken kunnen we deze vanuit meerdere plaatsen aanroepen (sluit knop en overlay klik).

Stap 4: Event listener voor overlay klik

Sluit modal bij klik op overlay (niet op content):

overlay.addEventListener('click', function(e) {
    if (e.target === overlay) closeModal();
});

Uitleg: e.target is het element waarop geklikt is. Als dat de overlay zelf is (niet de content), sluiten we de modal.

💻 Complete Code

const openBtn = document.getElementById('open-modal');
const overlay = document.getElementById('modal-overlay');
const closeBtn = document.getElementById('close-modal');

openBtn.addEventListener('click', function() {
    overlay.style.display = 'flex';
    document.body.style.overflow = 'hidden';
});

function closeModal() {
    overlay.style.display = 'none';
    document.body.style.overflow = 'auto';
}

closeBtn.addEventListener('click', closeModal);
overlay.addEventListener('click', function(e) {
    if (e.target === overlay) closeModal();
});

✅ Instructies:

  • Selecteer open knop, overlay en sluit knop
  • Toon overlay met display: flex bij open klik
  • Zet body overflow op hidden om scroll te voorkomen
  • Maak closeModal functie voor herbruikbaarheid
  • Controleer e.target om alleen overlay klik te detecteren

💡 Tips & Tricks

  • z-index: Zorg dat overlay hoge z-index heeft (bijv. 1000) om boven alles te zijn
  • e.target vs this: e.target is geklikt element, this is element met listener
  • stopPropagation: Gebruik e.stopPropagation() om event bubbling te stoppen
  • Escape toets: Voeg keyboard listener toe voor Escape toets om te sluiten

⚠️ Veelvoorkomende Fouten

Fout 1: Vergeten overflow te resetten

// ❌ FOUT - Pagina blijft niet scrollbaar
function closeModal() {
    overlay.style.display = 'none'; // Vergeet overflow!
}

// ✅ GOED - Reset overflow
function closeModal() {
    overlay.style.display = 'none';
    document.body.style.overflow = 'auto';
}

Fout 2: Modal sluit bij klik op content

// ❌ FOUT - Sluit altijd, ook bij klik op content
overlay.addEventListener('click', closeModal);

// ✅ GOED - Controleer target
overlay.addEventListener('click', function(e) {
    if (e.target === overlay) closeModal();
});

Fout 3: Verkeerde display waarde

// ❌ FOUT - Content niet gecentreerd
overlay.style.display = 'block';

// ✅ GOED - Flex voor centrering
overlay.style.display = 'flex'; // Met align-items en justify-content

🚀 Uitbreidingsmogelijkheden

Probeer deze uitdagingen om je vaardigheden te verbeteren:

  • Escape toets: Sluit modal met Escape toets
  • Animatie: Voeg fade-in animatie toe bij openen
  • Focus trap: Houd focus binnen modal (accessibility)
  • Meerdere modals: Ondersteun meerdere modals tegelijk
  • Form in modal: Voeg formulier toe met validatie

Voorbeeld: Escape Toets

document.addEventListener('keydown', function(e) {
    if (e.key === 'Escape' && overlay.style.display === 'flex') {
        closeModal();
    }
});

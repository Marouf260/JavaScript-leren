📝 Stap-voor-Stap Uitleg

Stap 1: Klasse-Gebaseerde Particles

We gebruiken een JavaScript Class voor elke particle. Elk particle heeft zijn eigen positie, snelheid, kleur en gedrag. We maken een array van honderden instanties!

class Particle {
    constructor() {
        this.x = Math.random() * canvas.width;
        this.vx = (Math.random() * speed) - speed / 2;
    }
    update() {
        this.x += this.vx;
        // Muisinteractie, muurbotsing, enz.
    }
}

Stap 2: Verbindingslijnen

We vergelijken elke particle met elke andere (O(n²) complexiteit). Als de afstand kleiner is dan linkDistance, tekenen we een lijn met opaciteit gebaseerd op afstand.

✅ Jouw Opdracht

  • Beweeg je muis over het canvas - particles wijken uit!
  • Klik op het canvas voor een plaatselijke explosie!
  • Gebruik de sliders om aantallen, snelheid en verbindingsafstand te regelen!

🚀 Uitbreidingsmogelijkheden

  • Kleurverloop: Kleur de verbindingslijnen met een gradiënt gebaseerd op de particlekleuren!

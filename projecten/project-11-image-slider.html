Project 11: Image Slider

Advanced

Maak een image slider met vorige/volgende knoppen.

📚 Wat ga je leren?

In dit project leer je werken met arrays, animaties en timing functies. Je leert:

  • Arrays gebruiken om meerdere images op te slaan
  • setTimeout gebruiken voor animatie timing
  • CSS transitions gebruiken voor smooth effecten
  • Modulo operator voor cyclische navigatie
  • Functies maken voor herbruikbare code

🎯 Belangrijke Concepten

  • setTimeout() - Voert code uit na een vertraging (in milliseconden)
  • CSS transitions - Animaties tussen CSS waarden
  • opacity - CSS eigenschap voor transparantie (0-1)
  • Modulo met negatieve getallen - (index - 1 + length) % length voor achteruit
  • Arrow functions - Korte functie syntax: () => {}

📝 Stap-voor-Stap Uitleg

Stap 1: Array met images en state

Maak array met image URLs en index variabele:

const images = [
    'https://via.placeholder.com/500x300/3498db/ffffff?text=Image+1',
    'https://via.placeholder.com/500x300/e74c3c/ffffff?text=Image+2',
    'https://via.placeholder.com/500x300/2ecc71/ffffff?text=Image+3',
    'https://via.placeholder.com/500x300/f39c12/ffffff?text=Image+4'
];
let currentIndex = 0;

Uitleg: Array bevat alle image URLs. currentIndex houdt bij welke image momenteel getoond wordt.

Stap 2: showImage functie met fade effect

Maak functie die image toont met fade animatie:

function showImage(index) {
    img.style.opacity = '0';
    setTimeout(() => {
        img.src = images[index];
        img.style.opacity = '1';
    }, 200);
    img.style.transition = 'opacity 0.3s';
}

Uitleg: Eerst fade out (opacity 0), wacht 200ms, verander src, dan fade in. Transition zorgt voor smooth animatie.

Stap 3: Vorige knop met modulo

Gebruik modulo voor cyclische navigatie achteruit:

prevBtn.addEventListener('click', function() {
    currentIndex = (currentIndex - 1 + images.length) % images.length;
    showImage(currentIndex);
});

Uitleg: (currentIndex - 1 + images.length) zorgt dat negatieve getallen positief worden voordat modulo wordt toegepast.

Stap 4: Volgende knop

Gebruik normale modulo voor vooruit navigatie:

nextBtn.addEventListener('click', function() {
    currentIndex = (currentIndex + 1) % images.length;
    showImage(currentIndex);
});

Uitleg: Bij laatste image (index 3) wordt (3 + 1) % 4 = 0, dus gaat terug naar eerste.

💻 Complete Code

const images = [
    'https://via.placeholder.com/500x300/3498db/ffffff?text=Image+1',
    'https://via.placeholder.com/500x300/e74c3c/ffffff?text=Image+2',
    'https://via.placeholder.com/500x300/2ecc71/ffffff?text=Image+3',
    'https://via.placeholder.com/500x300/f39c12/ffffff?text=Image+4'
];
let currentIndex = 0;
const img = document.getElementById('slider-image');
const prevBtn = document.getElementById('prev-btn');
const nextBtn = document.getElementById('next-btn');

function showImage(index) {
    img.style.opacity = '0';
    setTimeout(() => {
        img.src = images[index];
        img.style.opacity = '1';
    }, 200);
    img.style.transition = 'opacity 0.3s';
}

prevBtn.addEventListener('click', function() {
    currentIndex = (currentIndex - 1 + images.length) % images.length;
    showImage(currentIndex);
});

nextBtn.addEventListener('click', function() {
    currentIndex = (currentIndex + 1) % images.length;
    showImage(currentIndex);
});

✅ Instructies:

  • Maak array met image URLs
  • Maak currentIndex variabele voor state
  • Maak showImage functie met fade animatie
  • Gebruik setTimeout voor timing
  • Gebruik modulo voor cyclische navigatie

💡 Tips & Tricks

  • setTimeout timing: 200ms is goed voor fade, pas aan naar voorkeur
  • Transition: Zet transition op element, niet in setTimeout
  • Modulo uitleg: (index - 1 + length) % length werkt voor negatieve getallen
  • Preload images: Preload images voor snellere transitions

⚠️ Veelvoorkomende Fouten

Fout 1: Vergeten transition te zetten

// ❌ FOUT - Geen animatie, abrupte wijziging
img.style.opacity = '0';
img.src = images[index]; // Vergeet transition!

// ✅ GOED - Zet transition
img.style.transition = 'opacity 0.3s';
img.style.opacity = '0';

Fout 2: Verkeerde modulo voor achteruit

// ❌ FOUT - Negatieve modulo werkt niet zoals verwacht
currentIndex = (currentIndex - 1) % images.length; // Bij 0 wordt -1!

// ✅ GOED - Voeg length toe voor positief resultaat
currentIndex = (currentIndex - 1 + images.length) % images.length;

Fout 3: Vergeten src te wijzigen

// ❌ FOUT - Alleen opacity wijzigt, image blijft hetzelfde
img.style.opacity = '1'; // Vergeet img.src!

// ✅ GOED - Wijzig src in setTimeout
setTimeout(() => {
    img.src = images[index];
    img.style.opacity = '1';
}, 200);

🚀 Uitbreidingsmogelijkheden

Probeer deze uitdagingen om je vaardigheden te verbeteren:

  • Dots indicator: Voeg dots toe die huidige image tonen
  • Auto-play: Laat slider automatisch doorlopen
  • Keyboard navigatie: Gebruik pijltjestoetsen
  • Touch gestures: Voeg swipe functionaliteit toe
  • Thumbnail gallery: Voeg kleine thumbnails toe onder slider

Voorbeeld: Auto-play

let autoPlayInterval;

function startAutoPlay() {
    autoPlayInterval = setInterval(() => {
        currentIndex = (currentIndex + 1) % images.length;
        showImage(currentIndex);
    }, 3000); // Elke 3 seconden
}

function stopAutoPlay() {
    clearInterval(autoPlayInterval);
}

startAutoPlay();

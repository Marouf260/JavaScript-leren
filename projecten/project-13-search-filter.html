Project 13: Search Filter

Advanced

Filter een lijst op basis van zoekinput.

📚 Wat ga je leren?

In dit project leer je werken met string matching en real-time filtering. Je leert:

  • toLowerCase() gebruiken voor case-insensitive zoeken
  • includes() gebruiken om te controleren of tekst bevat
  • forEach gebruiken om over NodeList te itereren
  • Real-time filtering implementeren
  • Display eigenschap gebruiken om elementen te verbergen

🎯 Belangrijke Concepten

  • toLowerCase() - Converteert tekst naar kleine letters
  • includes() - Controleert of string een substring bevat
  • textContent - Eigenschap om tekst van element te lezen
  • forEach() - Itereert over array/NodeList
  • Real-time filtering - Filteren tijdens typen

📝 Stap-voor-Stap Uitleg

Stap 1: Elementen selecteren

Selecteer input en alle items die gefilterd moeten worden:

const searchInput = document.getElementById('search-input');
const items = document.querySelectorAll('.filter-item');

Uitleg: querySelectorAll geeft NodeList met alle elementen die de class hebben.

Stap 2: Input event listener

Luister naar input events voor real-time filtering:

searchInput.addEventListener('input', function() {
    const searchTerm = this.value.toLowerCase();
    // Filter logica
});

Uitleg: input event gaat af bij elke toetsaanslag. toLowerCase() maakt zoeken case-insensitive.

Stap 3: Filter logica met forEach

Itereer over items en controleer of ze matchen:

items.forEach(item => {
    const text = item.textContent.toLowerCase();
    if (text.includes(searchTerm)) {
        item.style.display = 'block';
    } else {
        item.style.display = 'none';
    }
});

Uitleg: Voor elk item controleren we of de tekst de zoekterm bevat. Zo ja: tonen, zo nee: verbergen.

💻 Complete Code

const searchInput = document.getElementById('search-input');
const items = document.querySelectorAll('.filter-item');

searchInput.addEventListener('input', function() {
    const searchTerm = this.value.toLowerCase();
    
    items.forEach(item => {
        const text = item.textContent.toLowerCase();
        if (text.includes(searchTerm)) {
            item.style.display = 'block';
        } else {
            item.style.display = 'none';
        }
    });
});

✅ Instructies:

  • Selecteer search input en alle filter items
  • Voeg input event listener toe
  • Converteer zoekterm naar lowercase
  • Gebruik forEach om over items te itereren
  • Gebruik includes() om te controleren of tekst matcht
  • Wijzig display op basis van match

💡 Tips & Tricks

  • Case-insensitive: Altijd toLowerCase() gebruiken voor beide strings
  • includes() vs indexOf(): includes() is moderner en leesbaarder dan indexOf() !== -1
  • Performance: Voor grote lijsten, gebruik debouncing om niet bij elke toets te filteren
  • Highlight matches: Markeer gevonden tekst met span elementen

⚠️ Veelvoorkomende Fouten

Fout 1: Vergeten toLowerCase

// ❌ FOUT - Case-sensitive zoeken
const searchTerm = this.value; // "Appel" matcht niet met "appel"
if (text.includes(searchTerm)) { ... }

// ✅ GOED - Case-insensitive
const searchTerm = this.value.toLowerCase();
const text = item.textContent.toLowerCase();
if (text.includes(searchTerm)) { ... }

Fout 2: Vergeten display te resetten

// ❌ FOUT - Items blijven verborgen na lege zoekterm
if (text.includes(searchTerm)) {
    item.style.display = 'block';
} // Vergeet else!

// ✅ GOED - Reset altijd
if (text.includes(searchTerm)) {
    item.style.display = 'block';
} else {
    item.style.display = 'none';
}

Fout 3: Verkeerde event type

// ❌ FOUT - Alleen bij blur, niet real-time
searchInput.addEventListener('blur', ...);

// ✅ GOED - Real-time met input event
searchInput.addEventListener('input', ...);

🚀 Uitbreidingsmogelijkheden

Probeer deze uitdagingen om je vaardigheden te verbeteren:

  • Debouncing: Wacht tot gebruiker stopt met typen voordat je filtert
  • Highlight matches: Markeer gevonden tekst in items
  • Meerdere filters: Filter op meerdere criteria tegelijk
  • Sorting: Sorteer resultaten op relevantie
  • No results message: Toon bericht wanneer niets gevonden wordt

Voorbeeld: Debouncing

let timeout;
searchInput.addEventListener('input', function() {
    clearTimeout(timeout);
    timeout = setTimeout(() => {
        // Filter logica hier
    }, 300); // Wacht 300ms
});

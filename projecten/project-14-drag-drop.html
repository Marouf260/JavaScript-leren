Project 14: Drag & Drop

Advanced

Maak elementen die je kunt slepen en neerzetten.

📚 Wat ga je leren?

In dit project leer je werken met HTML5 Drag and Drop API. Je leert:

  • draggable attribuut gebruiken
  • Drag events gebruiken (dragstart, dragover, drop, dragend)
  • dataTransfer gebruiken om data te verplaatsen
  • preventDefault() gebruiken om drop te allowen
  • Visual feedback geven tijdens drag operatie

🎯 Belangrijke Concepten

  • draggable="true" - HTML attribuut om element draggable te maken
  • dragstart - Event wanneer drag begint
  • dragover - Event tijdens drag over drop zone
  • drop - Event wanneer element wordt neergezet
  • dataTransfer - Object om data te verplaatsen tussen drag en drop

📝 Stap-voor-Stap Uitleg

Stap 1: Draggable elementen maken

Voeg draggable attribuut toe aan HTML elementen:

<div class="draggable" draggable="true">Item 1</div>

Uitleg: draggable="true" maakt element sleepbaar. Zonder dit attribuut werkt drag & drop niet.

Stap 2: dragstart event - Data opslaan

Sla data op in dataTransfer wanneer drag begint:

item.addEventListener('dragstart', function(e) {
    e.dataTransfer.setData('text/html', this.outerHTML);
    this.style.opacity = '0.5';
});

Uitleg: setData slaat HTML op. outerHTML geeft volledige HTML van element inclusief element zelf.

Stap 3: dragover event - Drop toestaan

Voorkom standaard gedrag om drop mogelijk te maken:

target.addEventListener('dragover', function(e) {
    e.preventDefault();
    this.style.borderColor = '#4CAF50';
});

Uitleg: Standaard staat browser drop niet toe. preventDefault() maakt drop mogelijk.

Stap 4: drop event - Data ophalen

Haal data op en voeg toe aan drop zone:

target.addEventListener('drop', function(e) {
    e.preventDefault();
    const data = e.dataTransfer.getData('text/html');
    this.insertAdjacentHTML('beforeend', data);
});

Uitleg: getData haalt opgeslagen data op. insertAdjacentHTML voegt HTML toe.

💻 Complete Code

const draggables = document.querySelectorAll('.draggable');
const target = document.getElementById('target');

draggables.forEach(item => {
    item.addEventListener('dragstart', function(e) {
        e.dataTransfer.setData('text/html', this.outerHTML);
        this.style.opacity = '0.5';
    });
    
    item.addEventListener('dragend', function() {
        this.style.opacity = '1';
    });
});

target.addEventListener('dragover', function(e) {
    e.preventDefault();
    this.style.borderColor = '#4CAF50';
});

target.addEventListener('dragleave', function() {
    this.style.borderColor = '#ccc';
});

target.addEventListener('drop', function(e) {
    e.preventDefault();
    const data = e.dataTransfer.getData('text/html');
    this.insertAdjacentHTML('beforeend', data);
    this.style.borderColor = '#ccc';
    
    // Verwijder origineel element
    const dragged = document.querySelector('.draggable[style*="opacity: 0.5"]');
    if (dragged) dragged.remove();
});

✅ Instructies:

  • Voeg draggable="true" toe aan elementen
  • Voeg dragstart listener toe met setData
  • Voeg dragover listener toe met preventDefault
  • Voeg drop listener toe met getData
  • Gebruik dragend voor visual feedback reset

💡 Tips & Tricks

  • preventDefault is cruciaal: Zonder preventDefault() werkt drop niet
  • dataTransfer types: Gebruik 'text/html' voor HTML, 'text/plain' voor tekst
  • Visual feedback: Verander opacity of border tijdens drag voor betere UX
  • dragend event: Reset visual feedback wanneer drag eindigt

⚠️ Veelvoorkomende Fouten

Fout 1: Vergeten preventDefault bij dragover

// ❌ FOUT - Drop werkt niet
target.addEventListener('dragover', function(e) {
    this.style.borderColor = '#4CAF50'; // Vergeet preventDefault!
});

// ✅ GOED - Voorkom standaard gedrag
target.addEventListener('dragover', function(e) {
    e.preventDefault();
    this.style.borderColor = '#4CAF50';
});

Fout 2: Vergeten draggable attribuut

// ❌ FOUT - Element is niet draggable
<div class="draggable">Item</div> // Vergeet draggable="true"!

// ✅ GOED - Voeg attribuut toe
<div class="draggable" draggable="true">Item</div>

Fout 3: Verkeerde data type

// ❌ FOUT - Type moet overeenkomen
e.dataTransfer.setData('text/html', data);
const data = e.dataTransfer.getData('text/plain'); // Verkeerd type!

// ✅ GOED - Gebruikzelfde type
e.dataTransfer.setData('text/html', data);
const data = e.dataTransfer.getData('text/html');

🚀 Uitbreidingsmogelijkheden

Probeer deze uitdagingen om je vaardigheden te verbeteren:

  • Drag preview: Pas drag preview aan met setDragImage()
  • Meerdere drop zones: Maak meerdere zones waar items kunnen worden neergezet
  • Sortable list: Laat items binnen lijst herordenen
  • Touch support: Voeg touch events toe voor mobiele devices
  • Undo functionaliteit: Bewaar geschiedenis voor undo/redo

Voorbeeld: Custom Drag Preview

item.addEventListener('dragstart', function(e) {
    const dragImage = this.cloneNode(true);
    dragImage.style.opacity = '0.5';
    e.dataTransfer.setDragImage(dragImage, 0, 0);
});

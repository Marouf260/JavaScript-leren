Project 15: Infinite Scroll

Advanced

Laad meer content automatisch wanneer je scrollt.

📚 Wat ga je leren?

In dit project leer je werken met scroll events en scroll berekeningen. Je leert:

  • Scroll events gebruiken voor detectie
  • Scroll properties gebruiken (scrollTop, scrollHeight, clientHeight)
  • Berekenen wanneer gebruiker bijna onderaan is
  • Dynamisch content toevoegen
  • Loading states beheren

🎯 Belangrijke Concepten

  • scrollTop - Afstand van bovenkant tot zichtbaar gebied
  • scrollHeight - Totale hoogte van scrollbaar content
  • clientHeight - Zichtbare hoogte van element
  • scroll event - Event dat afgaat tijdens scrollen
  • Threshold - Afstand tot einde voordat nieuwe content laadt

📝 Stap-voor-Stap Uitleg

Stap 1: Scroll properties begrijpen

Leer de drie belangrijke scroll properties:

const scrollTop = container.scrollTop; // Hoe ver gescrolled
const scrollHeight = container.scrollHeight; // Totale hoogte
const clientHeight = container.clientHeight; // Zichtbare hoogte

Uitleg: scrollTop + clientHeight = waar je bent. scrollHeight = waar einde is. Wanneer ze bijna gelijk zijn, ben je bijna onderaan.

Stap 2: Scroll event listener

Luister naar scroll events:

container.addEventListener('scroll', function() {
    const scrollTop = this.scrollTop;
    const scrollHeight = this.scrollHeight;
    const clientHeight = this.clientHeight;
    // Check logica
});

Uitleg: Scroll event gaat af bij elke scroll beweging. We berekenen of gebruiker bijna onderaan is.

Stap 3: Check of bijna onderaan

Controleer of gebruiker binnen threshold is:

if (scrollTop + clientHeight >= scrollHeight - 10) {
    // Laad meer content
}

Uitleg: -10 is threshold. Wanneer scrollTop + clientHeight binnen 10px van scrollHeight is, laad meer.

Stap 4: Dynamisch items toevoegen

Voeg nieuwe items toe met setTimeout voor loading effect:

loading.style.display = 'block';
setTimeout(() => {
    for (let i = 0; i < 3; i++) {
        itemCount++;
        const item = document.createElement('div');
        item.textContent = 'Item ' + itemCount;
        container.appendChild(item);
    }
    loading.style.display = 'none';
}, 500);

Uitleg: setTimeout simuleert loading tijd. In echte app zou dit een API call zijn.

💻 Complete Code

const container = document.getElementById('scroll-container');
const loading = document.getElementById('loading');
let itemCount = 3;

container.addEventListener('scroll', function() {
    const scrollTop = this.scrollTop;
    const scrollHeight = this.scrollHeight;
    const clientHeight = this.clientHeight;
    
    if (scrollTop + clientHeight >= scrollHeight - 10) {
        loading.style.display = 'block';
        
        setTimeout(() => {
            for (let i = 0; i < 3; i++) {
                itemCount++;
                const item = document.createElement('div');
                item.className = 'scroll-item';
                item.style.cssText = 'padding: 1rem; margin: 0.5rem 0; background: #f0f0f0; border-radius: 4px;';
                item.textContent = 'Item ' + itemCount;
                container.appendChild(item);
            }
            loading.style.display = 'none';
        }, 500);
    }
});

✅ Instructies:

  • Selecteer scroll container en loading indicator
  • Voeg scroll event listener toe
  • Bereken scrollTop, scrollHeight en clientHeight
  • Check of scrollTop + clientHeight >= scrollHeight - threshold
  • Toon loading, voeg items toe, verberg loading

💡 Tips & Tricks

  • Threshold waarde: 10-50px is goed. Te klein = te vaak laden, te groot = te laat laden
  • Debouncing: Voorkom meerdere loads tegelijk met een flag
  • Performance: Scroll events gaan vaak af, gebruik requestAnimationFrame voor betere performance
  • End detection: Check of er nog meer content is voordat je laadt

⚠️ Veelvoorkomende Fouten

Fout 1: Verkeerde scroll berekening

// ❌ FOUT - Verkeerde vergelijking
if (scrollTop >= scrollHeight) { ... } // Werkt niet!

// ✅ GOED - Gebruik scrollTop + clientHeight
if (scrollTop + clientHeight >= scrollHeight - 10) { ... }

Fout 2: Meerdere loads tegelijk

// ❌ FOUT - Kan meerdere keren laden
container.addEventListener('scroll', function() {
    if (nearBottom) loadMore(); // Geen flag!
});

// ✅ GOED - Gebruik loading flag
let isLoading = false;
if (nearBottom && !isLoading) {
    isLoading = true;
    loadMore();
}

Fout 3: Vergeten loading te verbergen

// ❌ FOUT - Loading blijft zichtbaar
loading.style.display = 'block';
loadItems(); // Vergeet loading.style.display = 'none'!

// ✅ GOED - Verberg na laden
loading.style.display = 'block';
loadItems();
loading.style.display = 'none';

🚀 Uitbreidingsmogelijkheden

Probeer deze uitdagingen om je vaardigheden te verbeteren:

  • Intersection Observer: Gebruik moderne API in plaats van scroll events
  • Virtual scrolling: Render alleen zichtbare items voor betere performance
  • Pull to refresh: Voeg pull-to-refresh functionaliteit toe
  • End message: Toon bericht wanneer alle content geladen is
  • Error handling: Behandel errors bij het laden van content

Voorbeeld: Intersection Observer

const observer = new IntersectionObserver((entries) => {
    if (entries[0].isIntersecting) {
        loadMore();
    }
});

const sentinel = document.createElement('div');
container.appendChild(sentinel);
observer.observe(sentinel);

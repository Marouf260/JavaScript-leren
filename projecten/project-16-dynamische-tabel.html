Project 16: Dynamische Tabel

Expert

Maak een tabel met sorteren, filteren en paginatie.

📚 Wat ga je leren?

In dit project leer je werken met complexe data manipulatie en tabel operaties. Je leert:

  • Array.sort() gebruiken voor sorteren
  • Array.from() gebruiken om NodeList naar Array te converteren
  • isNaN() gebruiken om te controleren of waarde nummer is
  • localeCompare() gebruiken voor string vergelijking
  • Complexe state management met meerdere functies

🎯 Belangrijke Concepten

  • Array.sort() - Sorteert array op basis van compare functie
  • Array.from() - Converteert NodeList/iterable naar Array
  • isNaN() - Controleert of waarde geen nummer is
  • localeCompare() - Vergelijkt strings met locale-aware sorting
  • cells[index] - Eigenschap om specifieke cel in rij te krijgen

📝 Stap-voor-Stap Uitleg

Stap 1: Headers selecteren en state

Selecteer headers en maak state object voor sort richtingen:

const headers = table.querySelectorAll('th');
let sortDirection = {}; // Object om richting per kolom bij te houden

Uitleg: sortDirection object houdt bij welke kolom welke richting heeft (true = oplopend, false = aflopend).

Stap 2: Event listener op headers

Voeg click listener toe aan elke header:

headers.forEach((header, index) => {
    header.addEventListener('click', function() {
        // Sorteer logica
    });
});

Uitleg: forEach geeft index parameter. Deze gebruiken we om te weten welke kolom gesorteerd moet worden.

Stap 3: Rijen converteren naar Array

Converteer NodeList naar Array voor sorteren:

const tbody = table.querySelector('tbody');
const rows = Array.from(tbody.querySelectorAll('tr'));

Uitleg: Array.from() is nodig omdat sort() alleen op arrays werkt, niet op NodeLists.

Stap 4: Sort functie met type detectie

Maak sort functie die nummers en strings behandelt:

rows.sort((a, b) => {
    const aText = a.cells[index].textContent.trim();
    const bText = b.cells[index].textContent.trim();
    
    if (!isNaN(aText) && !isNaN(bText)) {
        return sortDirection[index] ? aText - bText : bText - aText;
    }
    
    return sortDirection[index] 
        ? aText.localeCompare(bText)
        : bText.localeCompare(aText);
});

Uitleg: isNaN() controleert of tekst nummer is. Voor nummers gebruiken we -, voor strings localeCompare().

Stap 5: Rijen opnieuw toevoegen

Voeg gesorteerde rijen opnieuw toe aan tbody:

rows.forEach(row => tbody.appendChild(row));

Uitleg: appendChild verplaatst elementen (niet kopieert). Rijen worden in nieuwe volgorde geplaatst.

💻 Complete Code

const table = document.getElementById('data-table');
const searchInput = document.getElementById('table-search');
const headers = table.querySelectorAll('th');
let sortDirection = {};

headers.forEach((header, index) => {
    header.addEventListener('click', function() {
        const tbody = table.querySelector('tbody');
        const rows = Array.from(tbody.querySelectorAll('tr'));
        
        sortDirection[index] = !sortDirection[index];
        
        rows.sort((a, b) => {
            const aText = a.cells[index].textContent.trim();
            const bText = b.cells[index].textContent.trim();
            
            if (!isNaN(aText) && !isNaN(bText)) {
                return sortDirection[index] ? aText - bText : bText - aText;
            }
            
            return sortDirection[index] 
                ? aText.localeCompare(bText)
                : bText.localeCompare(aText);
        });
        
        rows.forEach(row => tbody.appendChild(row));
    });
});

searchInput.addEventListener('input', function() {
    const searchTerm = this.value.toLowerCase();
    const rows = table.querySelectorAll('tbody tr');
    
    rows.forEach(row => {
        const text = row.textContent.toLowerCase();
        row.style.display = text.includes(searchTerm) ? '' : 'none';
    });
});

✅ Instructies:

  • Selecteer tabel headers en maak sortDirection object
  • Voeg click listeners toe aan headers met index
  • Converteer rijen naar Array met Array.from()
  • Gebruik sort() met compare functie
  • Controleer type met isNaN() voor juiste sortering
  • Voeg gesorteerde rijen opnieuw toe

💡 Tips & Tricks

  • Array.from() is belangrijk: NodeLists hebben geen sort() methode
  • isNaN() check: Controleer beide waarden voordat je nummer sortering gebruikt
  • localeCompare(): Beter dan > voor strings omdat het locale-aware is
  • appendChild verplaatst: Element wordt verwijderd van oude locatie en toegevoegd aan nieuwe

⚠️ Veelvoorkomende Fouten

Fout 1: Vergeten Array.from()

// ❌ FOUT - NodeList heeft geen sort()
const rows = tbody.querySelectorAll('tr');
rows.sort(...); // Error!

// ✅ GOED - Converteer naar Array
const rows = Array.from(tbody.querySelectorAll('tr'));
rows.sort(...);

Fout 2: Verkeerde type check

// ❌ FOUT - typeof check werkt niet voor strings die nummers zijn
if (typeof aText === 'number') { ... }

// ✅ GOED - Gebruik isNaN()
if (!isNaN(aText) && !isNaN(bText)) { ... }

Fout 3: Vergeten sortDirection te toggle

// ❌ FOUT - Richting verandert niet
rows.sort(...); // Vergeet sortDirection[index] = !sortDirection[index]

// ✅ GOED - Toggle richting
sortDirection[index] = !sortDirection[index];
rows.sort(...);

Voorbeeld: Paginatie

let currentPage = 1;
const itemsPerPage = 10;

function showPage(page) {
    const start = (page - 1) * itemsPerPage;
    const end = start + itemsPerPage;
    rows.forEach((row, index) => {
        row.style.display = (index >= start && index < end) ? '' : 'none';
    });
}

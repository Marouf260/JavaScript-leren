Project 17: WYSIWYG Editor

Expert

Maak een rich text editor met formatting opties.

📚 Wat ga je leren?

In dit project leer je werken met contentEditable en document.execCommand. Je leert:

  • contentEditable gebruiken om elementen bewerkbaar te maken
  • document.execCommand() gebruiken voor text formatting
  • Focus management voor betere UX
  • Dynamisch knoppen genereren
  • Rich text editing implementeren

🎯 Belangrijke Concepten

  • contentEditable - HTML attribuut om element bewerkbaar te maken
  • document.execCommand() - API voor text formatting (deprecated maar nog gebruikt)
  • focus() - Methode om focus naar element te verplaatsen
  • Selection API - Modern alternatief voor execCommand
  • Rich text editing - Tekst bewerken met formatting

📝 Stap-voor-Stap Uitleg

Stap 1: contentEditable element maken

Maak div met contentEditable attribuut:

<div id="editor-content" contenteditable="true">Type hier...</div>

Uitleg: contenteditable="true" maakt element bewerkbaar zoals een textarea, maar met HTML formatting mogelijk.

Stap 2: Format knoppen met data attributen

Maak knoppen met data-command attributen:

<button class="format-btn" data-command="bold">B</button>
<button class="format-btn" data-command="italic">I</button>

Uitleg: data-command bevat de execCommand naam die uitgevoerd moet worden.

Stap 3: execCommand gebruiken

Voer formatting command uit bij klik:

formatBtns.forEach(btn => {
    btn.addEventListener('click', function() {
        const command = this.dataset.command;
        document.execCommand(command, false, null);
        content.focus();
    });
});

Uitleg: execCommand(command, showUI, value). showUI=false betekent geen dialoog, value=null betekent geen extra waarde.

Stap 4: Focus behouden

Zet focus terug op editor na formatting:

content.focus();

Uitleg: Focus zorgt dat gebruiker direct verder kan typen zonder eerst te klikken.

💻 Complete Code

const toolbar = document.getElementById('editor-toolbar');
const content = document.getElementById('editor-content');
const formatBtns = toolbar.querySelectorAll('.format-btn');

formatBtns.forEach(btn => {
    btn.addEventListener('click', function() {
        const command = this.dataset.command;
        document.execCommand(command, false, null);
        content.focus();
    });
});

✅ Instructies:

  • Maak div met contenteditable="true"
  • Maak toolbar met knoppen en data-command attributen
  • Selecteer alle format knoppen
  • Voeg click listeners toe die execCommand uitvoeren
  • Zet focus terug op editor na formatting

💡 Tips & Tricks

  • execCommand is deprecated: Gebruik Selection API voor moderne implementaties
  • Beschikbare commands: bold, italic, underline, justifyLeft, justifyCenter, insertUnorderedList
  • Selection check: Controleer of tekst geselecteerd is voordat je format toepast
  • Undo/redo: Gebruik execCommand('undo') en execCommand('redo')

⚠️ Veelvoorkomende Fouten

Fout 1: Vergeten contentEditable attribuut

// ❌ FOUT - Element is niet bewerkbaar
<div id="editor-content">Type hier...</div> // Vergeet contenteditable!

// ✅ GOED - Voeg attribuut toe
<div id="editor-content" contenteditable="true">Type hier...</div>

Fout 2: Verkeerde execCommand syntax

// ❌ FOUT - Verkeerde parameters
document.execCommand('bold'); // Werkt maar niet standaard

// ✅ GOED - Gebruik alle parameters
document.execCommand('bold', false, null);

Fout 3: Vergeten focus te zetten

// ❌ FOUT - Gebruiker moet handmatig klikken
document.execCommand('bold', false, null); // Vergeet focus!

// ✅ GOED - Zet focus
document.execCommand('bold', false, null);
content.focus();

🚀 Uitbreidingsmogelijkheden

Probeer deze uitdagingen om je vaardigheden te verbeteren:

  • Selection API: Gebruik moderne Selection API in plaats van execCommand
  • Color picker: Voeg tekst kleur en achtergrond kleur toe
  • Link insertie: Voeg dialoog toe om links in te voegen
  • Image upload: Laat gebruikers afbeeldingen uploaden
  • Export HTML: Exporteer content als HTML

Voorbeeld: Link Insertie

function insertLink() {
    const url = prompt('Voer URL in:');
    if (url) {
        document.execCommand('createLink', false, url);
    }
}

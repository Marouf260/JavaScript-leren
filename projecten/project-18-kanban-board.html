Project 18: Kanban Board

Expert

Maak een Kanban board met drag & drop functionaliteit.

📚 Wat ga je leren?

In dit project leer je een complexe applicatie bouwen met drag & drop tussen meerdere containers. Je leert:

  • Drag & drop tussen meerdere drop zones
  • Event listeners toevoegen aan dynamisch gemaakte elementen
  • State synchroniseren tussen verschillende containers
  • Complexe DOM manipulatie met meerdere elementen
  • Herbruikbare drag & drop logica

🎯 Belangrijke Concepten

  • Multiple drop zones - Meerdere containers die items kunnen ontvangen
  • Event delegation - Event listeners op parent voor dynamische kinderen
  • insertAdjacentHTML - Voegt HTML toe op specifieke positie
  • Dynamic event binding - Event listeners toevoegen aan nieuwe elementen
  • State persistence - State behouden tussen operaties

📝 Stap-voor-Stap Uitleg

Stap 1: Cards en columns selecteren

Selecteer alle draggable cards en drop zones:

const cards = document.querySelectorAll('.kanban-card');
const columns = document.querySelectorAll('.kanban-column');

Uitleg: We hebben beide NodeLists nodig: cards om te slepen en columns als drop zones.

Stap 2: Drag events op cards

Voeg dragstart en dragend toe aan elke card:

cards.forEach(card => {
    card.addEventListener('dragstart', function(e) {
        e.dataTransfer.setData('text/html', this.outerHTML);
        this.style.opacity = '0.5';
    });
    
    card.addEventListener('dragend', function() {
        this.style.opacity = '1';
    });
});

Uitleg: dragstart slaat HTML op, dragend reset visual feedback.

Stap 3: Drop events op columns

Voeg dragover, dragleave en drop toe aan elke column:

columns.forEach(column => {
    column.addEventListener('dragover', function(e) {
        e.preventDefault();
        this.style.border = '2px dashed #4CAF50';
    });
    
    column.addEventListener('drop', function(e) {
        e.preventDefault();
        const data = e.dataTransfer.getData('text/html');

Uitleg: Elke column kan items ontvangen. insertAdjacentHTML voegt nieuwe card toe.

Stap 4: Event listeners toevoegen aan nieuwe cards

Wanneer nieuwe card wordt toegevoegd, voeg listeners toe:

const newCard = this.querySelector('.kanban-card:last-child');
if (newCard) {
    newCard.addEventListener('dragstart', function(e) {
        e.dataTransfer.setData('text/html', this.outerHTML);
        this.style.opacity = '0.5';
    });
}

Uitleg: Dynamisch gemaakte elementen hebben geen event listeners. We moeten ze handmatig toevoegen.

💻 Complete Code

const cards = document.querySelectorAll('.kanban-card');
const columns = document.querySelectorAll('.kanban-column');

cards.forEach(card => {
    card.addEventListener('dragstart', function(e) {
        e.dataTransfer.setData('text/html', this.outerHTML);
        this.style.opacity = '0.5';
    });
    
    card.addEventListener('dragend', function() {
        this.style.opacity = '1';
    });
});

columns.forEach(column => {
    column.addEventListener('dragover', function(e) {
        e.preventDefault();
        this.style.border = '2px dashed #4CAF50';
    });
    
    column.addEventListener('dragleave', function() {
        this.style.border = 'none';
    });
    
    column.addEventListener('drop', function(e) {
        e.preventDefault();
        const data = e.dataTransfer.getData('text/html');
        this.insertAdjacentHTML('beforeend', data);
        this.style.border = 'none';
        
        const dragged = document.querySelector('.kanban-card[style*="opacity: 0.5"]');
        if (dragged) dragged.remove();
        
        const newCard = this.querySelector('.kanban-card:last-child');
        if (newCard) {
            newCard.addEventListener('dragstart', function(e) {
                e.dataTransfer.setData('text/html', this.outerHTML);
                this.style.opacity = '0.5';
            });
        }
    });
});

✅ Instructies:

  • Selecteer alle cards en columns
  • Voeg dragstart en dragend toe aan cards
  • Voeg dragover, dragleave en drop toe aan columns
  • Gebruik insertAdjacentHTML om card toe te voegen
  • Verwijder originele card en voeg listeners toe aan nieuwe

💡 Tips & Tricks

  • insertAdjacentHTML positions: 'beforeend' = einde, 'afterbegin' = begin, 'beforebegin' = voor element, 'afterend' = na element
  • Event delegation: Overweeg event delegation op parent om listeners niet handmatig toe te voegen
  • State tracking: Bewaar welke card in welke column zit voor localStorage
  • Visual feedback: Verander border kleur tijdens drag voor betere UX

⚠️ Veelvoorkomende Fouten

Fout 1: Vergeten listeners toe te voegen aan nieuwe cards

// ❌ FOUT - Nieuwe card is niet draggable
this.insertAdjacentHTML('beforeend', data); // Vergeet listeners!

// ✅ GOED - Voeg listeners toe
this.insertAdjacentHTML('beforeend', data);
const newCard = this.querySelector('.kanban-card:last-child');
if (newCard) {
    newCard.addEventListener('dragstart', ...);
}

Fout 2: Vergeten originele card te verwijderen

// ❌ FOUT - Card bestaat nu twee keer
this.insertAdjacentHTML('beforeend', data); // Vergeet verwijderen!

// ✅ GOED - Verwijder origineel
const dragged = document.querySelector('.kanban-card[style*="opacity: 0.5"]');
if (dragged) dragged.remove();

Fout 3: Verkeerde insertAdjacentHTML positie

// ❌ FOUT - Card wordt op verkeerde plek geplaatst
this.insertAdjacentHTML('beforebegin', data); // Buiten column!

// ✅ GOED - Gebruik beforeend voor einde
this.insertAdjacentHTML('beforeend', data); // Aan einde van column

function saveBoard() {
    const boardData = {};

        boardData[columnId] = Array.from(column.querySelectorAll('.kanban-card')).map(card => card.textContent);
    });
    localStorage.setItem('kanbanBoard', JSON.stringify(boardData));
}

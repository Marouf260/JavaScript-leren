Project 19: Real-time Chat

Expert

Maak een chat interface met berichten die automatisch verschijnen.

📚 Wat ga je leren?

In dit project leer je een real-time chat interface bouwen. Je leert:

  • Dynamisch berichten toevoegen met timestamps
  • Auto-scroll implementeren naar laatste bericht
  • Date object gebruiken voor timestamps
  • innerHTML gebruiken voor HTML content
  • Simuleren van real-time updates met setTimeout

🎯 Belangrijke Concepten

  • Date object - JavaScript object voor datum en tijd
  • toLocaleTimeString() - Formatteert tijd naar locale string
  • scrollHeight - Totale scrollbare hoogte
  • scrollTop - Huidige scroll positie
  • Template literals - Strings met ${variabele} voor dynamische content

📝 Stap-voor-Stap Uitleg

Stap 1: addMessage functie maken

Maak functie die berichten toevoegt met timestamp:

function addMessage(text, user = 'Jij') {
    const messageDiv = document.createElement('div');
    messageDiv.className = 'message';
    
    const time = new Date().toLocaleTimeString();
    messageDiv.innerHTML = `<strong>${user}:</strong> ${text} <span>(${time})</span>`;
    
    messagesContainer.appendChild(messageDiv);
    messagesContainer.scrollTop = messagesContainer.scrollHeight;
}

Uitleg: Template literal maakt HTML string. scrollTop = scrollHeight scrollt naar beneden.

Stap 2: sendMessage functie

Maak functie die bericht verzendt en input leegt:

function sendMessage() {
    if (chatInput.value.trim() !== '') {
        addMessage(chatInput.value);
        chatInput.value = '';
    }
}

Uitleg: Valideer input, voeg bericht toe, leeg input veld.

Stap 3: Simuleer antwoord

Gebruik setTimeout om automatisch antwoord te simuleren:

setTimeout(() => {
    addMessage('Bedankt voor je bericht!', 'Bot');
}, 1000);

Uitleg: setTimeout simuleert server response. In echte app zou dit WebSocket of polling zijn.

💻 Complete Code

const messagesContainer = document.getElementById('chat-messages');
const chatInput = document.getElementById('chat-input');
const sendBtn = document.getElementById('chat-send');

function addMessage(text, user = 'Jij') {
    const messageDiv = document.createElement('div');
    messageDiv.className = 'message';
    messageDiv.style.cssText = 'margin-bottom: 0.5rem; padding: 0.5rem; background: #e3f2fd; border-radius: 4px;';
    
    const time = new Date().toLocaleTimeString();
    messageDiv.innerHTML = `<strong>${user}:</strong> ${text} <span style="font-size: 0.75rem; color: #666;">(${time})</span>`;
    
    messagesContainer.appendChild(messageDiv);
    messagesContainer.scrollTop = messagesContainer.scrollHeight;
}

function sendMessage() {
    if (chatInput.value.trim() !== '') {
        addMessage(chatInput.value);
        chatInput.value = '';
        
        setTimeout(() => {
            addMessage('Bedankt voor je bericht!', 'Bot');
        }, 1000);
    }
}

sendBtn.addEventListener('click', sendMessage);
chatInput.addEventListener('keypress', function(e) {
    if (e.key === 'Enter') sendMessage();
});

  • Gebruik Date object voor tijd
  • Gebruik innerHTML voor HTML content
  • Zet scrollTop op scrollHeight voor auto-scroll
  • Maak sendMessage functie die input valideert
  • Simuleer antwoord met setTimeout

💡 Tips & Tricks

  • scrollHeight: Totale hoogte inclusief niet-zichtbare content
  • toLocaleTimeString(): Formatteert tijd volgens locale instellingen
  • Template literals: Gebruik backticks voor strings met variabelen
  • innerHTML vs textContent: innerHTML voor HTML, textContent voor alleen tekst

⚠️ Veelvoorkomende Fouten

Fout 1: Vergeten auto-scroll

// ❌ FOUT - Nieuwe berichten zijn niet zichtbaar
messagesContainer.appendChild(messageDiv); // Vergeet scroll!

// ✅ GOED - Scroll naar beneden
messagesContainer.appendChild(messageDiv);
messagesContainer.scrollTop = messagesContainer.scrollHeight;

Fout 2: Verkeerde scroll property

// ❌ FOUT - scrollHeight is read-only
messagesContainer.scrollHeight = 1000; // Werkt niet!

// ✅ GOED - Zet scrollTop
messagesContainer.scrollTop = messagesContainer.scrollHeight;

Fout 3: Vergeten input te valideren

// ❌ FOUT - Lege berichten kunnen worden verzonden
addMessage(chatInput.value); // Vergeet trim()!

// ✅ GOED - Valideer eerst
if (chatInput.value.trim() !== '') {
    addMessage(chatInput.value);
}

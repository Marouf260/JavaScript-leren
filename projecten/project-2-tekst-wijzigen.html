Project 2: Tekst Wijzigen

Beginner

Wijzig tekst in een element wanneer je op een knop klikt.

📚 Wat ga je leren?

In dit project leer je hoe je tekst dynamisch kunt wijzigen in een HTML element. Je leert:

  • Elementen selecteren met getElementById
  • Event listeners toevoegen aan knoppen
  • De textContent eigenschap gebruiken om tekst te wijzigen
  • Het verschil tussen textContent en innerHTML begrijpen

🎯 Belangrijke Concepten

  • textContent - Eigenschap om alleen tekst te wijzigen (veilig tegen XSS)
  • innerHTML - Eigenschap om HTML code toe te voegen (kan gevaarlijk zijn)
  • getElementById() - Selecteert een element op basis van zijn ID attribuut
  • addEventListener('click') - Luistert naar klik events op een element
  • Event-driven programming - Code die reageert op gebruikersacties

📝 Stap-voor-Stap Uitleg

Stap 1: Elementen selecteren

Eerst moeten we de elementen uit de HTML halen die we nodig hebben:

const tekst = document.getElementById('change-text');
const knop = document.getElementById('change-btn');

Uitleg: We gebruiken getElementById om beide elementen te vinden. De IDs moeten exact overeenkomen met de IDs in je HTML.

Stap 2: Event Listener toevoegen

We luisteren naar klik events op de knop:

knop.addEventListener('click', function() {
    // Code die uitgevoerd wordt bij klik
});

Uitleg: Wanneer de gebruiker op de knop klikt, wordt de functie uitgevoerd die we als tweede parameter hebben gegeven.

Stap 3: Tekst wijzigen

In de event listener functie wijzigen we de tekst:

knop.addEventListener('click', function() {
    tekst.textContent = 'Tekst is gewijzigd!';
});

Uitleg: textContent overschrijft alle tekst binnen het element. Dit is veiliger dan innerHTML omdat het geen HTML code uitvoert.

💻 Complete Code

// Stap 1: Selecteer elementen
const tekst = document.getElementById('change-text');
const knop = document.getElementById('change-btn');

// Stap 2 & 3: Event listener met tekst wijziging
knop.addEventListener('click', function() {
    tekst.textContent = 'Tekst is gewijzigd!';
});

✅ Instructies:

  • Selecteer het tekst element met getElementById
  • Selecteer de knop element
  • Voeg een click event listener toe aan de knop
  • Wijzig de textContent bij klik

💡 Tips & Tricks

  • textContent vs innerHTML: Gebruik textContent voor gewone tekst. Gebruik alleen innerHTML als je HTML tags nodig hebt, maar wees voorzichtig met gebruikersinput!
  • Meerdere wijzigingen: Je kunt de tekst meerdere keren wijzigen door verschillende knoppen te maken of een toggle functie te gebruiken.
  • Dynamische tekst: Je kunt variabelen gebruiken: tekst.textContent = 'Hallo ' + naam;
  • Conditional tekst: Gebruik if/else om verschillende teksten te tonen op basis van condities.

⚠️ Veelvoorkomende Fouten

Fout 1: Verkeerde ID naam

// ❌ FOUT - ID bestaat niet
const tekst = document.getElementById('text'); // Moet 'change-text' zijn

// ✅ GOED - Correcte ID
const tekst = document.getElementById('change-text');

Fout 2: Element selecteren voordat het bestaat

// ❌ FOUT - Script staat boven HTML elementen
const tekst = document.getElementById('change-text'); // null!

// ✅ GOED - Script staat onderaan of gebruikt DOMContentLoaded
// Script tag onderaan HTML of:
document.addEventListener('DOMContentLoaded', function() {
    const tekst = document.getElementById('change-text');
});

Fout 3: Vergeten event listener toe te voegen

// ❌ FOUT - Geen event listener, tekst wijzigt niet
tekst.textContent = 'Nieuwe tekst'; // Wordt direct uitgevoerd

// ✅ GOED - Event listener zorgt voor reactie op klik
knop.addEventListener('click', function() {
    tekst.textContent = 'Nieuwe tekst';
});

🚀 Uitbreidingsmogelijkheden

Probeer deze uitdagingen om je vaardigheden te verbeteren:

  • Toggle functionaliteit: Wissel tussen twee teksten bij elke klik
  • Input veld: Laat gebruiker tekst invoeren en toon die in het element
  • Meerdere elementen: Wijzig meerdere elementen tegelijk met één klik
  • Animatie: Voeg een fade-in effect toe wanneer tekst wijzigt
  • Geschiedenis: Bewaar vorige teksten en voeg een "undo" knop toe

Voorbeeld: Toggle Functionaliteit

let isGewijzigd = false;

knop.addEventListener('click', function() {
    if (isGewijzigd) {
        tekst.textContent = 'Originele tekst';
    } else {
        tekst.textContent = 'Tekst is gewijzigd!';
    }
    isGewijzigd = !isGewijzigd;
});

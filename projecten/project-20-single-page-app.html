Project 20: Single Page App

Expert

Maak een complete SPA met routing en state management.

📚 Wat ga je leren?

In dit project leer je een complete Single Page Application bouwen. Je leert:

  • Client-side routing implementeren
  • History API gebruiken voor URL updates
  • popstate event gebruiken voor browser navigatie
  • State management tussen pagina's
  • Hash-based routing implementeren

🎯 Belangrijke Concepten

  • Hash routing - Routing op basis van URL hash (#home, #about)
  • History API - pushState() om URL te updaten zonder reload
  • popstate event - Event bij browser back/forward
  • window.location.hash - Eigenschap om hash te lezen
  • SPA (Single Page App) - App die zonder page reload werkt

📝 Stap-voor-Stap Uitleg

Stap 1: showPage functie maken

Maak functie die pagina toont en andere verbergt:

function showPage(pageId) {
    pages.forEach(page => {
        page.style.display = 'none';
    });
    
    const targetPage = document.getElementById(pageId + '-page');
    if (targetPage) {
        targetPage.style.display = 'block';
    }
}

Uitleg: Verberg eerst alle pagina's, toon dan alleen de gevraagde pagina. pageId wordt gecombineerd met '-page' voor ID.

Stap 2: Active link styling

Update styling van actieve navigatie link:

links.forEach(link => {
    link.style.fontWeight = 'normal';
    link.style.color = '#333';
});

const activeLink = document.querySelector(`[href="#${pageId}"]`);
if (activeLink) {
    activeLink.style.fontWeight = 'bold';
    activeLink.style.color = '#F7DF1E';
}

Uitleg: Reset eerst alle links, markeer dan actieve link met andere styling.

Stap 3: Link click handlers

Voeg click listeners toe die preventDefault gebruiken:

links.forEach(link => {
    link.addEventListener('click', function(e) {
        e.preventDefault();
        const pageId = this.getAttribute('href').substring(1);
        showPage(pageId);
        window.history.pushState({page: pageId}, '', '#' + pageId);
    });
});

Uitleg: preventDefault() voorkomt standaard link gedrag. substring(1) verwijdert #. pushState() update URL.

Stap 4: Browser back/forward handling

Luister naar popstate voor browser navigatie:

window.addEventListener('popstate', function(e) {
    const pageId = e.state ? e.state.page : 'home';
    showPage(pageId);
});

Uitleg: popstate gaat af bij back/forward. e.state bevat data die we met pushState hebben opgeslagen.

Stap 5: Initial page load

Laad juiste pagina bij eerste load:

const initialPage = window.location.hash.substring(1) || 'home';
showPage(initialPage);

Uitleg: Lees hash uit URL. Als er geen hash is, gebruik 'home' als default.

💻 Complete Code

const links = document.querySelectorAll('.spa-link');
const pages = document.querySelectorAll('.spa-page');

function showPage(pageId) {
    pages.forEach(page => {
        page.style.display = 'none';
    });
    
    const targetPage = document.getElementById(pageId + '-page');
    if (targetPage) {
        targetPage.style.display = 'block';
    }
    
    links.forEach(link => {
        link.style.fontWeight = 'normal';
        link.style.color = '#333';
    });
    
    const activeLink = document.querySelector(`[href="#${pageId}"]`);
    if (activeLink) {
        activeLink.style.fontWeight = 'bold';
        activeLink.style.color = '#F7DF1E';
    }
}

links.forEach(link => {
    link.addEventListener('click', function(e) {
        e.preventDefault();
        const pageId = this.getAttribute('href').substring(1);
        showPage(pageId);
        window.history.pushState({page: pageId}, '', '#' + pageId);
    });
});

window.addEventListener('popstate', function(e) {
    const pageId = e.state ? e.state.page : 'home';
    showPage(pageId);
});

const initialPage = window.location.hash.substring(1) || 'home';
showPage(initialPage);

✅ Instructies:

  • Maak showPage functie die pagina's toont/verbergt
  • Update active link styling in showPage
  • Voeg click listeners toe met preventDefault
  • Gebruik pushState om URL te updaten
  • Luister naar popstate voor browser navigatie
  • Laad initial page op basis van hash

💡 Tips & Tricks

  • Hash vs History: Hash routing (#) werkt zonder server config, History API (/path) vereist server setup
  • pushState parameters: (state, title, url) - state wordt opgeslagen, title wordt vaak genegeerd
  • popstate state: e.state bevat data die je met pushState hebt opgeslagen
  • substring(1): Verwijdert eerste karakter (#) van hash

⚠️ Veelvoorkomende Fouten

Fout 1: Vergeten preventDefault

// ❌ FOUT - Pagina reload bij klik
link.addEventListener('click', function() {
    showPage(pageId); // Vergeet preventDefault!
});

// ✅ GOED - Voorkom standaard gedrag
link.addEventListener('click', function(e) {
    e.preventDefault();
    showPage(pageId);
});

Fout 2: Vergeten popstate handler

// ❌ FOUT - Back knop werkt niet
window.history.pushState(...); // Vergeet popstate listener!

// ✅ GOED - Luister naar popstate
window.addEventListener('popstate', function(e) {
    showPage(e.state.page);
});

Fout 3: Verkeerde hash parsing

// ❌ FOUT - Hash bevat nog steeds #
const pageId = window.location.hash; // "#home"
document.getElementById(pageId + '-page'); // "#home-page" bestaat niet!

// ✅ GOED - Verwijder #
const pageId = window.location.hash.substring(1); // "home"
document.getElementById(pageId + '-page'); // "home-page"

🚀 Uitbreidingsmogelijkheden

Probeer deze uitdagingen om je vaardigheden te verbeteren:

  • Route parameters: Ondersteun routes zoals #user/123
  • Route guards: Voorkom toegang tot bepaalde routes
  • Lazy loading: Laad pagina content alleen wanneer nodig
  • Transition animations: Voeg animaties toe bij route changes
  • Nested routes: Ondersteun geneste routes

Voorbeeld: Route Parameters

function parseRoute(hash) {
    const parts = hash.split('/');
    return {
        page: parts[0],
        params: parts.slice(1)
    };
}

const route = parseRoute(window.location.hash.substring(1));
// #user/123 wordt {page: 'user', params: ['123']}

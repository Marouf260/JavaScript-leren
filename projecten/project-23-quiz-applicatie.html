Project 23: Quiz Applicatie

Master

Bouw een interactieve quiz met statemanagement, voortgansgrafieken en directe feedback.

📝 Stap-voor-Stap Uitleg

Stap 1: De Vragen Data Opslaan

We creëren een betrouwbare structuur voor onze vragen door gebruik te maken van een Array met Objecten erin. Elk object bevat de vraag, antwoorden, en de index van het juiste antwoord.

const questions = [
    { q: "Wat is JS?", options: ["Auto", "Taal"], answer: 1 }
];

Uitleg: Omdat arrays 0-geïndexeerd zijn, verwijst answer: 1 naar "Taal".

Stap 2: State Management

Een quiz is een "stateful" app; hij moet onthouden waar we zijn en wat de score is.

let currentQuestion = 0;
let score = 0;
let canAnswer = true;

Uitleg: Telkens als de gebruiker een vraag voltooid, verhogen we currentQuestion en hergebruiken we in loadQuestion() hetzelfde DOM element, in plaats van 10 nieuwe pagina's te maken!

Stap 3: CSS Score Animatie

Op het eindscherm gebruiken we setInterval() gecombineerd met conic-gradient in CSS om een animerende ring te tonen.

let currentPerc = 0;
let interval = setInterval(() => {
    if (currentPerc >= percentage) clearInterval(interval);
    circle.style.background = `conic-gradient(#8e44ad ${currentPerc}%, #eee ${currentPerc}%)`;
    currentPerc++;
}, 15);

Uitleg: Door elke 15 milliseconden het percentage met +1 te verhogen en de background te updaten, wordt een "vollopend" taartdiagram gecreëerd.

💻 Code Snippet (Controleer Antwoord)

function checkAnswer(optDiv, index) {
    // Voorkom 2x klikken (Anti-Spam beveiliging)
    if (!canAnswer) return;
    canAnswer = false;

    const isCorrect = (index === questions[currentQuestion].answer);
    if (isCorrect) score++;

    const allOptions = optionsContainer.children;
    for (let i = 0; i < allOptions.length; i++) {
        if (i === questions[currentQuestion].answer) {
            allOptions[i].classList.add("correct");
        } else if (i === index && !isCorrect) {
            allOptions[i].classList.add("incorrect");
        }
    }
}

💡 Tips & Tricks

  • Waarom canAnswer toevoegen? Als je geen block neerzet, kan een gebruiker direct 4 keer op het de juiste antwoord klikken om 4 punten te genereren per vraag. Een boolean flag sluit de actie "op slot".
  • Dynamische Layouts: In plaats van voor 20 vragen, 20 verschillende HTML-pagina's of pop-ups op te bouwen, genereren wij 1 template en vervang je op de vlucht uitsluitend de text elementen. Veel sneller!

⚠️ Veelvoorkomende Fouten

Fout 1: Hardcoded DOM Elementen (Injevuld in HTML, niet in JS)

// ❌ FOUT - Alle opties in de HTML schrijven maakt het aanpassen van de quiz moeilijk.
<button id="opt1">Antwoord 1</button>

// ✅ GOED - Genereer knoppen dynamisch o.b.v. de array in JS.
q.options.forEach((opt, index) => {
    const optDiv = document.createElement("div");
    optDiv.className = "option";
    optDiv.innerHTML = `<span>${opt}</span>`;
    optionsContainer.appendChild(optDiv);
});

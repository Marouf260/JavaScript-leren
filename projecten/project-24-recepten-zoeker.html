Project 24: Recepten Zoeker

Master

Bouw een receptenzoekmachine met geavanceerde .filter() methoden in JavaScript.

📝 Stap-voor-Stap Uitleg

Stap 1: HTML Elementen aanmaken in Javascript met Template Literals

We creëren HTML voor onze recepten niet in index.html, maar met zogenaamde "Template Literals" (backticks `` ` ``) in JavaScript, waardoor we waarden dynamisch kunnen inkleuren.

const card = document.createElement('div');
card.className = 'recipe-card';
card.innerHTML = `
    <img src="${r.img}" alt="${r.title}" class="recipe-img">
    <h3 class="recipe-title">${r.title}</h3>
`;

Uitleg: Het gebruik van ${variabele} binnen backticks laat je direct variabelen en methodes uitvoeren ín een string!

Stap 2: .map() & .join() voor Geneste data (Tags)

Een recept kan meerdere tags hebben (bijv "Gezond", "Lunch"). We converteren de JavaScript array van strings naar een HTML string list.

${r.tags.map(t => `<span class="tag">${t}</span>`).join('')}

Uitleg: De .map() functie transformeert elke tag (bijv. "Lunch") in de gewenste HTML-tag. .join('') plakt al deze gemaakte HTML-tags naadloos zonder komma's aan elkaar.

Stap 3: Krachtig Zoeken met .filter()

Tijdens het typen, pakken we alle recepten en filteren we alleen degene die we willen overhouden en weergaven.

const query = searchInput.value.toLowerCase();
const filtered = mockRecipes.filter(r => 
    r.title.toLowerCase().includes(query) || 
    r.tags.some(t => t.toLowerCase().includes(query))
);

Uitleg: .includes() kijkt of de in-getypte tekst ergens in de titel of tags zit. Het gebruik van .some() controleert of in ieder geval één tag voldoet.

💻 Code Snippet (Real-Time Zoeken via "input" event)

function handleSearch() {
    const query = searchInput.value.toLowerCase();
    const filtered = mockRecipes.filter(r => 
        r.title.toLowerCase().includes(query) || 
        r.tags.some(t => t.toLowerCase().includes(query))
    );
    renderRecipes(filtered);
}

// De magie: Activeert IEDERE keer als een karakter wordt ingetypt/gewist
searchInput.addEventListener('input', handleSearch);
document.getElementById('searchBtn').addEventListener('click', handleSearch);

✅ Jouw Opdracht:

  • Typ direct 'Oven' of 'Lunch' in de zoekbalk hieronder in de demo.
  • Verwijder karakters met backspace en zie hoe recepten direct weer terugspringen.
  • Breid (lokaal in je eigen editor) het `mockRecipes` object uit met jouw lievelingsgerecht.

💡 Tips & Tricks

  • Waarom grid.innerHTML = ''? Om te voorkomen dat de resultaten van een nieuwe zoekopdracht gewoon onder de resultaten van een eerdere zoekopdracht worden geplakt, maken we eerst in renderRecipes() razendsnel de Grid leeg voordat de forEach wordt gestart.
  • Event 'input' vs 'change': Gebruik input in de plaats van change op tekstvelden omdat change pas afvuurt wanneer de gebruiker buiten het invoerveld klikt (blur).

🚀 Uitbreidingsmogelijkheden

Klaar voor meer? Probeer dit toe te voegen:

  • No Results feedback: We tonen al een tekst bij 0 results, voeg hier een icon (bijv. FontAwesome) of illustratie aan toe.
  • Debouncing: Voeg "debounce" logica toe zodat handleSearch() pas afvuurt als de gebruiker 300ms stopt met typen (handig bij zware applicaties of API requests).
  • Dropdown categorie filter: Voeg een <select> dropdown toe om snel te filteren op categorieën zoals "Diner", "Vis", "Lunch", of "Gezond".

Project 26: Muziek Speler

Master

Bouw een muziekspeler met draaiende platen covers, een voortgangsbalk en play/pauze commando's.

📝 Stap-voor-Stap Uitleg

Stap 1: De Audio API

Browsers hebben een ingebouwde speler voor geluid. Dit zit in de <audio> tag in je HTML, of je kan zoiets direct in JS maken met new Audio().

const audio = document.getElementById('audio');

if (isPlaying) {
    audio.pause();
} else {
    audio.play();
}

Uitleg: We roepen letterlijk .play() of .pause() aan op het DOM element! De browser regelt de rest.

Stap 2: De Voortgangsbalk (Tijd Updaten)

Terwijl de muziek speelt, zendt het audio element constant het timeupdate event uit. Hiermee kunnen we een visuele balk vullen.

function updateProgress(e) {
    const { duration, currentTime } = e.srcElement;
    const progressPercent = (currentTime / duration) * 100;
    
    progress.style.width = `${progressPercent}%`;
}

Uitleg: We berekenen de verstreken tijd t.o.v. de totale duur (uit het event properties), vertalen dat naar een percentage (0 tot 100) en passen daarmee de CSS width stapsgewijs aan.

Stap 3: De Klikbare Tijdlijn (Skippen)

Wanneer de gebruiker op een willekeurig punt op de lange balk tikt, spoelt de audio door naar dat moment.

function setProgress(e) {
    const width = this.clientWidth;
    const clickX = e.offsetX;
    
    audio.currentTime = (clickX / width) * audio.duration;
}

Uitleg: Als de balk 100px breed is, en je klikt op 50px (offsetX), dan zit je op exact 50%. We vertellen de audio.currentTime te springen naar de helft van de duration (totale tracklengte).

💻 Code Snippet (Visuele Muziek Synchronisatie)

// Als we op 'Play' klikken wekken we ook de interface tot leven.
function togglePlay() {
    if (isPlaying) {
        audio.pause();
        playBtn.innerText = '▶';
        cover.classList.remove('play'); // Stop de CD-animatie
    } else {
        audio.play();
        playBtn.innerText = '⏸';
        cover.classList.add('play'); // Start de 360-graden draai-animatie
    }
    isPlaying = !isPlaying;
}

✅ Jouw Opdracht:

  • Druk in de demo op de grote afspeelknop. Het liedje zal meteen afspelen (en de LP / cover image zal theatraal ronddraaien!).
  • Druk op "volgende" voor een andere titel (in deze demo blijft het audiosample hetzelfde bestand, maar reset de timer naar het begin).
  • Klik ergens op de grijze tijdlijn balk om in een vingerknip door het muziekstuk te skippen.

💡 Tips & Tricks

  • Muis-locatie e.offsetX: In muis events is offsetX relatief aan het blok waar je op klikte. Het vertelt je de muis positie binnen in de div, wat ideaal is voor berekeningen van o.a. balken en sliders.
  • CORS & URL-Restricties: In JS is het vaak verboden om media van andere sites/domeinen met JS te bedienen om copyright en beveiligingsredenen (CORS). Voeg crossorigin="anonymous" toe aan je <audio> of <img> tag in je HTML om permissie te forceren op openbare servers!

⚠️ Veelvoorkomende Fouten

Fout 1: Vroegtijdige start (NaN of Not a Number duration)

// ❌ FOUT - Als het zware audiobestand in de 1e milliseconde nog moet laden, is duration nul! Delen door nul crasht rekenformules.
const progressPercent = (currentTime / duration) * 100;

// ✅ GOED - Verzeker dat je weel hoelang het nummer daadwerkelijk is (voorkom een fatale crash).
// Een simpele 'return' sluit de functie direct The af the vroege stadium.
if (!duration) return;

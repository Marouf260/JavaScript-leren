📝 Stap-voor-Stap Uitleg

Stap 1: Zinnen opsplitsen in losse HTML Elementen

Om te weten of een gebruiker letter-voor-letter de tekst goed typt, moeten we de string omzetten naar individuele array-karakters met .split(''). Van elk karakter maken we een eigen <span>, waardoor we die apart rood of groen kunnen kleuren. 

const RANDOM_QUOTE = "De code compilet niet...";

RANDOM_QUOTE.split('').forEach(char => {
    const charSpan = document.createElement('span');
    charSpan.innerText = char;
    quoteDisplay.appendChild(charSpan);
});

Uitleg: We vullen de zwarte "terminal" box dynamisch letter-voor-letter met HTML Spans. Dit heet het prepareren / scaffolden van de UI (User Interface).

Stap 2: Live Vergelijken (Array tegen Array)

Elke keer als er getypt wordt (event: 'input'), splitsen we wat de gebruiker daadwerkelijk getypt heeft óók op in een nieuw array en vergelijken we de letter op index [0] met de doeltaal letter op index [0]. 

const arrayQuote = quoteDisplay.querySelectorAll('span');
const arrayValue = quoteInput.value.split('');

arrayQuote.forEach((charSpan, index) => {
    const character = arrayValue[index];
    if (character === charSpan.innerText) {
        charSpan.classList.add('correct');
    }
});

Uitleg: Door parallel te lopen over the array van de querySelectorAll, kunnen we live CSS classes toevoegen (rood/groen) gebaseerd op een if-else match structuur.

Stap 3: WPM Formule (Words per Minute)

Om te berekenen hoe snel de gebruiker is in "Words Per Minute", hebben we het aantal letters, seconden en een wiskunde formule nodig.

let cpm = Math.round((valLength / timeElapsed) * 60);
let wpm = Math.round(((valLength / 5) / timeElapsed) * 60);

Uitleg: In internationale typ-competities telt 1 woord gestandaardiseerd als 5 karakters. Dus: (aantal characters / 5) gedeeld door het aantal seconden dat is verstreken KEER 60 (maal 60, voor 1 minuut).

  • Typ nauwkeurig mee in het kader. Foute toetsaanslagen kleuren bloedrood.

💡 Tips & Tricks

  • Waarom Math.round? Het aantal typ-letters (bijv 38) gedeeld door seconden (bijv. 7) geeft vervelende ongenuanceerde decimale breuken (5.42857142857). Math.round zorgt direct voor schone, leesbare integers (5).
  • Waarom split('') met enkele quotes? De split() functie breekt een string op in stukjes. Als je niets meegeeft split(), houd je 1 massief blok over. Als je split(' ') - met spatie - gebruikt, dan breekt het de hele string in losse woorden op! Een lege string '' (niks) breekt het op EXACT elke letter!

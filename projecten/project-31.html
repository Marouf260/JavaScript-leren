Project 31: Parallax Scrolling

Advanced

Maak een adembenemend Parallax effect waarbij de achtergrond sneller of langzamer scrolt dan de voorgrond.

📝 Stap-voor-Stap Uitleg

Stap 1: De Scroll Listener

We binden een addEventListener('scroll') aan het element waarin we scrollen. In ons geval is dat het containerblok. Elke keer als de muis scrolt of je op een touchscreen swipet, wordt deze logica afgevuurd!

// Vang ELKE scroll-beweging op in het frame 
scrollArea.addEventListener('scroll', () => {
    // 1. Lees de huidige top-positie pixel
    const scrollPos = scrollArea.scrollTop;
    // 2. Start met de DOM transformaties (zie stap 2)
});

Uitleg: scrollTop is een fantastische property. Het vertelt ons in pure getallen (pixels) hoeveel het specifieke element naar beneden is gescrold. Bij de start (bovenaan het frame) is dit exact 0.

Stap 2: De Achtergrond Vertragen

Een "Parallax" effect onstaat als de achtergrond trager beweegt dan het perspectief van de voorgrond (onze witte boxen). We gebruiken CSS Transform om de achtergrond naar beneden (Y-as) te schuiven, gebaseerd op een wiskundig getal kleiner dan 1.

// De achtergrond scrolt SLECHTS de helft van de snelheid (0.5x) 
// mee naar beneden ten opzichte van onze echte scroll-positie.
pBg.style.transform = `translateY(${scrollPos * 0.5}px)`;

Uitleg: In modern JavaScript is style.transform = ... veel beter en sneller (qua prestaties) voor de browser elementen vergeleken met het live aanpassen van style.top of margin-waardes! Het activeert hardware-acceleratie op de processor van je computer (GPU compositing).

Stap 3: Diepte met Meerdere Snelheden

Niet alleen de achtergrond hoeft mee te doen aan Parallax! Om een magisch 3D perspectief te schetsen, pakken we alle individuele voorgrond-boxjes ("Zwevende elementen") met querySelectorAll(), en geven ze elk een ANDERE translateY factor in de for-loop!

pBoxes.forEach((box, index) => {
    // De index is 0, 1, 2 etc. De scroll factor verandert the uniek per box!
    const speed = 0.1 * (index + 1); 
    
    // Verplaats de box the the Y the as the  
    box.style.transform = `translateY(${scrollPos * speed}px)`;
});

Uitleg: Het the resultaat? De bovenste box beweegt in verhouding veel trager (0x) dan de onderste box in the array (bijv 0.3x). Dit the creëert eenthe prachtig visueel the zwaartekracht "zweef" effect the the op thethe ons the the dashboard-scherm the.

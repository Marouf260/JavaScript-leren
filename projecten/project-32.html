Project 32: Custom Video Player

Advanced

Verberg de standaard browser videocontroles en bouw je eigen state-of-the-art UI.

📝 Stap-voor-Stap Uitleg

Stap 1: De Standaard Controls Verbergen

Browsers (Safari, Chrome, Firefox) tonen normaal hun eigen video player interface. Die interface verschilt per browser. We voorkomen dit door eenvoudigweg de HTML controls property weg te laten uit de <video> tag!

<!-- Geen 'controls' attribuut = we bouwen zelf de UI! -->
<video id="vid" src="movie.mp4"></video>

Uitleg: Vanaf nu doet de video he-le-maal niks meer als de gebruiker erop klikt, tenzij we JavaScript inzetten om de afspeel logica the kapen!

Stap 2: Play & Pauze Logica bouwen

De HTML5 video API geeft JS directe properties zoals video.paused, en commandos zoals video.play() en video.pause().

togglePlay() {
    if (video.paused) {
        video.play();
        playBtn.innerHTML = '⏸';
    } else {
        video.pause();
        playBtn.innerHTML = '▶️';
    }
}

Uitleg: Deze functie binden we aan onze eigen HTML PLAY-knop én aan een klik op de video zelf. We toggelen tegelijkertijd het icoontje (Play vs Pauze)!

Stap 3: Het timeupdate Event

Hoe updatet de rode tijdbalk zich tijdens het afspelen van de video? De <video> HTML tag vuurt automatisch op de achtergrond het timeupdate event af. Tot wel meerdere keren per seconde!

video.addEventListener('timeupdate', updateProgress);

function updateProgress() {
    // (huidige tijd / Totale tijd) * 100 === Percentage!
    const percentage = (video.currentTime / video.duration) * 100;
    progressBar.style.width = `${percentage}%`;
}

Uitleg: Door die wiskundige deling te maken in updateProgress, kunnen we exact de CSS width van de rode progressiebalk updaten.

💻 Code Snippet (Tijd Formatteren)

// Computers denken in PURE secondes (bijv. 135 secondes). 
// Wij mensen willen zien: 02:15. Dan heb je wiskunde nodig!
function formatTime(seconds) {
    const min = Math.floor(seconds / 60); // 135 / 60 = 2 minuten
    const sec = Math.floor(seconds % 60); // Restwaarde = 15s
    
    // Voeg voorloopnullen ('0') toe als het getal onder de 10 is!
    return `${min < 10 ? '0' : ''}${min}:${sec < 10 ? '0' : ''}${sec}`;
}

✅ Jouw Opdracht:

  • Beweeg je muis over de video (hover!) om de controls te onthullen.
  • Druk op de play/pauze knop of klik ergens op de video zelf.
  • Klik ergens op de RODE progressiebalk om door de video te spoelen (skippen)!

💡 Tips & Tricks

  • timeupdate vs progress: In de code gebruiken we het timeupdate event om de balk in real-time te updaten. Het progress event wordt gevuurd door de browser als er nieuwe video data geladen/gebufferd wordt!
  • Scrubbing (Slepen op de Balk): Door een variabele mousedown op TRUE of FALSE te zetten in de muisevents, weten we of de gebruiker op de rode balk klikt óf aan het slepen is!

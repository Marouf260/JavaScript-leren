Project 33: 3D Tilt Hover Kaarten

Advanced

Bouw Premium 'Apple-style' 3D kaarten die roteren naar je muis toe met een licht-glans effect.

📝 Stap-voor-Stap Uitleg

Stap 1: HTML Box Boundaries Uitlezen

Voordat we berekeningen kunnen doen, moeten we de precieze dimensies van de gehoverde HTML kaart te weten komen. We gebruiken hier de professionele API getBoundingClientRect() voor.

card.addEventListener('mousemove', (e) => {
    // Wat is de X, Y, Width en Height van deze kaart in de browser?
    const rect = card.getBoundingClientRect();
    
    // Bereken de muis-positie puur BINNEN de kaart!
    const x = e.clientX - rect.left;
    const y = e.clientY - rect.top;
});

Uitleg: Als we de de links- en top-marge van het document van onze globale e.clientX en e.clientY aftrekken, weten we exact waar de muis op de HTML kaart zich bevindt!

Stap 2: Wiskunde voor 3D Rotatie

We willen de kaart kantelen naar DEZELFDE kant als waar de muis zich bevindt.

// De '10' is een dempingsfactor. Zonder dit draait de kaart TE SNEL!
const rotateY = -((x - rect.width / 2) / 10);
const rotateX = ((y - rect.height / 2) / 10);

card.style.transform = `rotateX(${rotateX}deg) rotateY(${rotateY}deg)`;

Uitleg: Door de breedte van de kaart door 2 te delen, weten we waar het middelpunt is. Als de muis links van het midden is, wordt x - width/2 een NEGATIEF getal. Dit zorgt ervoor dat de kaart naar links kantelt!

Stap 3: Het 'Glare' Effect (Lichtglans)

Een echte premium kaart heeft ook een lichtglans die over het scherm schiet als je de kaart beweegt! 

// Hoever zit de muis op de kaart in procenten? 
const glarePercent = (x / rect.width) * 100;

// Schuif de witte gradient over de kaart heen
glare.style.transform = `translateX(${glarePercent - 100}%)`;

Uitleg: .glare is een div met een witte linear-gradient die we met een translateX altijd mooi laten meeschuiven met de muis.

💻 Code Snippet (Reset upon Leave)

// Als de muis de kaart verlaat, moeten we het kaartje weer plat leggen!
card.addEventListener('mouseleave', () => {
    // Reset de rotatie naar 0 graden (plat)
    card.style.transform = `perspective(1000px) rotateX(0) rotateY(0)`;
    card.style.transition = `transform 0.5s ease`; // Zachte animatie terug!
});

✅ Jouw Opdracht:

  • Beweeg je muis over de twee kaarten hieronder.
  • Let op hoe de tekst "zweeft" bovenop de achtergrond!
  • Let ook op de witte "glans" (glare) die door de kaart schiet.

💡 Tips & Tricks

  • perspective(1000px): Zonder deze CSS eigenschap in de parent of direct in de kaart JS, krijg je geen 3D effect! Deze waarde bepaalt hoe "diep" de 3D inkijk is. Je kan de hoek extremer maken door dit naar 500px te veranderen.

⚠️ Veelvoorkomende Fouten

Fout 1: transition conflicts

// ❌ LET OP - Als je de CSS transition op de kaart altijd aan laat staan (bijv. transition: all 0.3s), dan zal de kaart langzamer de muis volgen. Dit veroorzaakt een wankelende, schokkende beweging. Daarom is card.style.transition = 'none' in de mousemove heel cruciaal!

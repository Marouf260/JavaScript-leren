Project 35: Circular Progress Bars

Advanced

Bouw prachtige, geanimeerde ronde voortgangsbalken. Ideaal voor dashboards en portfolio's!

📝 Stap-voor-Stap Uitleg

Stap 1: SVG Cirkels in HTML

Een ronde progressiebalk maken we niet met standaard <div> elementen, maar met Scalable Vector Graphics (SVG). We tekenen hiermee twee perfecte <circle> elementen over elkaar heen: een grijze achtergrondcirkel en een gekleurde voorgrondcirkel.

<!-- cx en cy bepalen het middelpunt, r is de straal (radius) -->
<svg>
    <circle class="circle-bg" cx="75" cy="75" r="60"></circle>
    <circle class="circle-stroke" cx="75" cy="75" r="60"></circle>
</svg>

Uitleg: In de CSS draaien we de hele SVG -90deg zodat de start van de voorgrondcirkel bovenaan (op 12 uur) begint in plaats van rechts (op 3 uur).

Stap 2: De SVG 'Dash' Truc

Hoe animeren we de lijn? We gebruiken de magische SVG stroke eigenschappen. Door de grens (stroke) van de cirkel in "streepjes" (dashes) op te delen, kunnen we één heel lang streepje maken dat evengroot is als de hele omtrek!

// 1. Bereken de Omtrek (2 * π * straal)
const circumference = 2 * Math.PI * radius;

// 2. Maak het "streepje" even lang als de hele cirkel!
circle.style.strokeDasharray = `\${circumference} \${circumference}`;

Uitleg: Nu is de cirkel technisch gezien één massief streepje. Door in stap 3 de basis (offset) van die streep te verschuiven, lijkt het of we de cirkel vullen of leegmaken!

Stap 3: Het Percentage Berekenen

Met de stroke-dashoffset verschuiven we het beginpunt van ons streepje we uit de vorige stap.

function setProgress(percent) {
    // Wiskunde: hoeveel offset is 25% "leegte"?
    const offset = circumference - (percent / 100) * circumference;
    
    // Dankzij CSS 'transition' verandert deze waarde super soepel!
    circle.style.strokeDashoffset = offset;
}

Uitleg: Als we percent = 100 doorgeven, wordt de offset 0 en is de hele cirkel te zien. Als percent = 0 is, is de offset gelijk aan de hele omtrek en verdwijnt de progressie uit beeld!

💻 Code Snippet (SetTimeout Delay)

// Als we de functie direct triggeren bij het openen van de pagina,
// ziet de gebruiker de CSS animatie niet opbouwen. Wachten is de sleutel!
setTimeout(() => {
    setProgress('react-circle', 'react-val', 75); // 75% vol
}, 300);

✅ Jouw Opdracht:

  • Bekijk de cirkels, wanneer ze inladen schieten ze mooi vloeiend naar de startwaardes!
  • Klik op "Genereer Random Data" om de balken vloeiend naar willekeurige nieuwe waardes te laten springen.
  • Merk op dat ook het percentage in het midden direct verandert.

💡 Tips & Tricks

  • stroke-linecap: round: Door deze simpele CSS regel mee te geven aan je SVG cirkel, krijgt het uiteinde van je progressiebalk een mooie zachte afronding in plaats van een harde rechte lijn. Ontzettend clean!
  • Dynamische Kleuren: In Javascript kun je eenvoudig met een if-statement checken: is de value onder de 50? Maak de circle.style.stroke dan oranje of rood.

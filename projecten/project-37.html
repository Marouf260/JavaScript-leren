Project 37: Kanban Board (Trello-stijl)

Advanced

Bouw een agile taakbeheer bord met pure HTML5 Drag & Drop API's.

📝 Stap-voor-Stap Uitleg

Stap 1: De HTML5 Drag & Drop API

Om een kaartje sleepbaar te maken, hoef je in HTML alleen maar draggable="true" toe te voegen aan het element. Vervolgens luisteren we in Javascript naar het moment dat de muis het element oppakt.

card.addEventListener('dragstart', (event) => {
    // 1. Geef het gesleepte element een doorzichtige CSS class
    card.classList.add('dragging');
    
    // 2. SLA HET ID OP in het "dataTransfer" geheugen van de browser!
    event.dataTransfer.setData('text/plain', card.id);
});

Uitleg: Het event dragstart triggert de seconde dat je klikt en beweegt. We moeten in dat event direct in de "DataTransfer" kluis van den browser opslaan *welk* specifieke kaartje (ID) verplaatst wordt!

Stap 2: De Kolom (Dropzone) Voorbereiden

De kolommen ("Te Doen", "Bezig") accepteren normaal gesproken geen dragged elementen. We moeten de browser expliciet vertellen dat er wél dropped mag worden d.m.v dragover.

kolom.addEventListener('dragover', (event) => {
    // ❌ CRUCIAAL: Voorkom het standaardgedrag van de browser (wat droppen verbiedt)
    event.preventDefault();
    
    // Visuele feedback dat het hier past
    kolom.classList.add('drag-over');
});

Uitleg: Zonder de magische e.preventDefault() in de dragover listener, zal de browser niet toelaten dat je hier iets laat vallen.

Stap 3: Het Taakje Laten Vallen (Drop)

Het moment dat je muisklikt loslaat, triggert het drop event op de kolom. Hier gaan we het ID uitlezen uit de kluis en het element appenden.

kolom.addEventListener('drop', (event) => {
    event.preventDefault();
    
    // 1. Lees het ID uit de dataTransfer kluis
    const gesleeptId = event.dataTransfer.getData('text/plain');
    
    // 2. Zoek het originele HTML element op 
    const kaartje = document.getElementById(gesleeptId);
    
    // 3. Plaats het in de nieuwe kolom!
    kolom.appendChild(kaartje);
});

Uitleg: Javascript appendChild() is zo slim dat het niet een kopie maakt, maar het element verplaatst. Het verwijdert de taak direct uit de oude kolom.

💻 Code Snippet (SetTimeout hack in Dragstart)

card.addEventListener('dragstart', () => {
    // Zonder deze setTimeout verdwijnt niet ALLEEN de kaart op de oude plek, maar OOK het drag-icoontje onder je muis! 
    setTimeout(() => card.style.display = 'none', 0);
});

✅ Jouw Opdracht:

  • Pak een bestaande taak op met je muis (houd muisknop ingedrukt).
  • Sleep de taak naar een andere kolom en laat los.
  • Klik op "+ Voeg taak toe" om een nieuwe taak toe te voegen.

💡 Tips & Tricks

  • Visuele Feedback: Het is belangrijk dat de gebruiker visuele feedback krijgt dat hij iets oppakt. Daarom maken we in de CSS de class dragging iets doorzichtig. Zo weet de gebruiker dat het element is opgepakt!
  • Vergeet dragleave niet: Zorg dat je ook een dragleave event toevoegt om de drag-over class te verwijderen als de gebruiker met de muis de kolom verlaat zonder te droppen.

⚠️ Veelvoorkomende Fouten

Fout 1: De "Ik kan niet droppen!" frustratie

Project 4: Show/Hide Element

Beginner

Toon of verberg een element met een knop.

📚 Wat ga je leren?

In dit project leer je hoe je elementen dynamisch kunt tonen en verbergen. Je leert:

  • De display CSS eigenschap gebruiken om zichtbaarheid te controleren
  • Boolean variabelen gebruiken om state bij te houden
  • Conditional logic (if/else) toepassen
  • Toggle functionaliteit implementeren

🎯 Belangrijke Concepten

  • element.style.display - CSS eigenschap om element te tonen ('block') of verbergen ('none')
  • Boolean variabelen - Variabelen die true of false zijn (waar of onwaar)
  • if/else statements - Conditionele logica om verschillende acties uit te voeren
  • Toggle pattern - Wisselen tussen twee states (zichtbaar/onzichtbaar)
  • ! operator - Logische NOT operator die true naar false maakt en vice versa

📝 Stap-voor-Stap Uitleg

Stap 1: Elementen selecteren

Selecteer het element dat je wilt tonen/verbergen en de knop:

const box = document.getElementById('toggle-box');
const knop = document.getElementById('toggle-btn');

Uitleg: We hebben beide elementen nodig: de box om te tonen/verbergen en de knop om de actie te triggeren.

Stap 2: State variabele maken

Maak een boolean variabele om bij te houden of het element zichtbaar is:

let isZichtbaar = true; // Begin zichtbaar

Uitleg: Deze variabele houdt bij of het element momenteel zichtbaar is. We beginnen met true omdat het element standaard zichtbaar is.

Stap 3: Event listener met conditional logic

Voeg een event listener toe die controleert of het element zichtbaar is:

knop.addEventListener('click', function() {
    if (isZichtbaar) {
        box.style.display = 'none';
        knop.textContent = 'Toon';
    } else {
        box.style.display = 'block';
        knop.textContent = 'Verberg/Toon';
    }
    isZichtbaar = !isZichtbaar;
});

Uitleg: Als isZichtbaar true is, verbergen we het element en updaten we de knop tekst. Anders tonen we het en updaten we de tekst. De ! operator wisselt de waarde.

💻 Complete Code

// Stap 1: Selecteer elementen
const box = document.getElementById('toggle-box');
const knop = document.getElementById('toggle-btn');

// Stap 2: State variabele
let isZichtbaar = true;

// Stap 3: Event listener met toggle logica
knop.addEventListener('click', function() {
    if (isZichtbaar) {
        box.style.display = 'none';
        knop.textContent = 'Toon';
    } else {
        box.style.display = 'block';
        knop.textContent = 'Verberg/Toon';
    }
    isZichtbaar = !isZichtbaar;
});

✅ Instructies:

  • Selecteer het element dat je wilt tonen/verbergen
  • Selecteer de toggle knop
  • Maak een boolean variabele om state bij te houden
  • Voeg event listener toe met if/else logica
  • Wijzig display eigenschap en update knop tekst

💡 Tips & Tricks

  • Alternatief met classList: Je kunt ook CSS classes gebruiken: box.classList.toggle('hidden') met CSS .hidden { display: none; }
  • display vs visibility: display: none verwijdert het element volledig, visibility: hidden verbergt het maar behoudt ruimte
  • Kortere toggle: Je kunt ook direct togglen: box.style.display = box.style.display === 'none' ? 'block' : 'none'
  • Opacity animatie: Gebruik opacity en transition voor smooth fade effecten

⚠️ Veelvoorkomende Fouten

Fout 1: Vergeten state bij te werken

// ❌ FOUT - State wordt niet geüpdatet, werkt maar één keer
if (isZichtbaar) {
    box.style.display = 'none';
} // Vergeet isZichtbaar = !isZichtbaar

// ✅ GOED - Update state na wijziging
if (isZichtbaar) {
    box.style.display = 'none';
}
isZichtbaar = !isZichtbaar;

Fout 2: Verkeerde display waarde

// ❌ FOUT - 'hidden' is geen geldige display waarde
box.style.display = 'hidden'; // Werkt niet!

// ✅ GOED - Gebruik 'none' om te verbergen
box.style.display = 'none'; // Verbergt element
box.style.display = 'block'; // Toont element

Fout 3: Vergeten display type te specificeren

// ❌ FOUT - Element wordt niet getoond als het inline was
box.style.display = ''; // Lege string werkt niet

// ✅ GOED - Specificeer display type
box.style.display = 'block'; // Of 'flex', 'inline-block', etc.

🚀 Uitbreidingsmogelijkheden

Probeer deze uitdagingen om je vaardigheden te verbeteren:

  • Fade animatie: Gebruik opacity en transition voor smooth fade in/out
  • Slide animatie: Animeer height of transform voor slide effect
  • Meerdere elementen: Toggle meerdere elementen tegelijk met één knop
  • Accordeon: Sluit andere elementen wanneer je een nieuwe opent
  • Keyboard shortcut: Voeg Escape toets toe om te verbergen

Voorbeeld: Fade Animatie

// CSS toevoegen:
// .fade { transition: opacity 0.3s; }
// .hidden { opacity: 0; pointer-events: none; }

box.classList.add('fade');
knop.addEventListener('click', function() {
    box.classList.toggle('hidden');
});

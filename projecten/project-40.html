Project 40: Custom Select Dropdown

Advanced

Vervang de generieke browser <select> box door een op maat gemaakte, styling-vriendelijke UI dropdown.

📝 Stap-voor-Stap Uitleg

Stap 1: De HTML Structuur

We bouwen de 'select' na met gewone <div> en <ul> elementen. Hierdoor hebben we volledige vrijheid over de CSS styling, in tegenstelling tot de standaard browser-gebonden <select> lijst.

<!-- De hoofd knop die de geselecteerde waarde toont -->
<div class="select-btn" id="select-btn">
    <span id="s-text">Selecteer je Social Media</span>
    <i>⬇️</i>
</div>

<!-- Het dropdown menu (standaard onzichtbaar via CSS opacity: 0) -->
<ul class="options">
    <li class="option">...</li>
</ul>

Stap 2: Openen en Sluiten

Als we op de hoofdknop klikken, willen we het menu tonen of verbergen (togglen). Ook handelen we de kliks op de individuele opties direct af.

// Toggle het CSS 'active' class (toont dropdown & draait pijltje)
selectBtn.addEventListener('click', () => {
    selectMenu.classList.toggle('active');
});

// Voor elke lay-out optie in het lijstje...
options.forEach(option => {
    option.addEventListener('click', () => {
        // 1. Haal de tekst op uit de span in deze li
        let geselecteerdeTekst = option.querySelector('.option-text').innerText;
        // 2. Plak tekst in de hoofdknop!
        hoofdKnopSpantag.innerText = geselecteerdeTekst;
        // 3. Klap menu dicht!
        selectMenu.classList.remove('active');
    });
});

Stap 3: CSS Class Toggling (Optioneel: Highlights)

Naast de tekst vervangen, wil je vaak in de UI laten zien 'dit item in de lijst is the active selection'. Dat doen we door CSS classes zoals .selected te togglen.

// Verwijder eerst van ALLE opties de 'selected' class 
options.forEach(opt => opt.classList.remove('selected'));

// Voeg daarna toe aan de specifieke optie waarop geklikt is! 
option.classList.add('selected');

💻 Code Snippet (Buiten de Dropdown Sluiten)

// 🎯 PRO-TIP: De gebruiker verwacht dat een dropdown dichtgaat als hij/zij ergens ANDERS klikt.

// Luister naar ELKE muis-klik op het complete document
document.addEventListener('click', (event) => {
    // Als doel van de "click" NIET in ons "-custom-select-" element zit...
    if (!selectMenu.contains(event.target)) {
        // Klap de dropdown in!
        selectMenu.classList.remove('active');
    }
});

✅ Jouw Opdracht:

  • Klik op "Selecteer je Social Media" in het witte vak.
  • Kies een optie en zie hoe deze de waarde aanneemt en het menu sluit.
  • Open het menu nog eens, en klik vervolgens op een willekeurige zin *buiten* the Select Box (bijvoorbeeld ergens in deze tekst). Hierdoor klapt hij netjes dicht!

💡 Tips & Tricks

  • Verborgen Inputs: Als je deze custom select wilt gebruiken in een ECHT formulier dat naar een server verzonden wordt, moet je er een onzichtbare input aan toevoegen. Maak een <input type="hidden" name="social"> en werk in Javascript telkens diens value bij als de gebruiker een item kiest. Clicks alleen worden niet meegestuurd in formulieren!

⚠️ Veelvoorkomende Fouten

Fout 1: Event Bubbling

// ❌ LET OP - De 'outside-click' listener reageert op elke klik in het document. Verzeker jezelf ervan dat je klik op de select knop niet dubbel triggert via Event Bubbling! In dit specifieke script lost de `contains` techniek dat elegant voor je op. Als je echter met `stopPropagation()` werkt kan dat de the outside-click listener mogelijk breken!

Project 41: Wachtwoord Generator

Data Logic

Bouw een tool die complexe, gerandomiseerde strings genereert op basis van gebruikersinstellingen.

📝 Stap-voor-Stap Uitleg

Stap 1: De "Pool" (Karakter Vijver) Bouwen

We maken voor elke soort (hoofdletters, cijfers, symbolen) een lange string met alle mogelijke karakters. Als de gebruiker een checkbox aanvinkt, gooien we die string in de grote passwordList vijver.

let passwordList = '';

// Als checkbox 'lower' aan staat (true), voeg a-z toe aan de pool
if (lower) passwordList += 'abcdefghijklmnopqrstuvwxyz';
if (number) passwordList += '0123456789';

// Als beide aanstaan is de vijver nu: 'abcdefghijklmnopqrstuvwxyz0123456789'

Stap 2: Hengelen met Random()

Nu we een mooie vijver hebben, gaan we er met een for loop net zovaak uit hengelen (random karakter pakken) als de lengte (slider) die de gebruiker wil.

let gegenereerdWachtwoord = '';

// Loop 'length' aantal keer (bijv. 12 keer)
for (let i = 0; i < length; i++) {
    // Kies een willekeurig getal tussen 0 en de lengte van de vijver
    const randomIndex = Math.floor(Math.random() * passwordList.length);
    
    // Plak die specifieke letter vast aan het wachtwoord antwoord
    gegenereerdWachtwoord += passwordList[randomIndex];
}

Stap 3: Het Klembord (Clipboard API)

Het gegenereerde wachtwoord overtypen is niet te doen. We gebruiken een ingebouwde browserfunctie (API) om het direct naar het Klembord van de gebruiker (CTRL+C) te sturen!

// Belangrijk: De Clipboard API werkt via 'Promises' (Asynchroon)
// Het duurt een milliseconde om naar de OS te praten. .then() voert pas uit als het OS 'OK' zegt!
navigator.clipboard.writeText(wachtwoord).then(() => {
    alert("Gekopieerd!"); 
});

Uitleg: We geven ook direct feedback in de knop (kleurtje wijzigen) zodat de gebruiker wéét dat de actie is geslaagd. Dat is goede UX design!

✅ Jouw Opdracht:

  • Zet opties aan of uit (Kleine letters, Cijfers, Symbolen) met de checkboxes.
  • Schuif de "Lengte" slider omhoog of omlaag en zie het getal veranderen.
  • Druk op Genereer Wachtwoord en bekijk het resultaat! Gebruik tenslotte de "Kopieer" knop en plak het (CTRL+V) ergens.

💡 Tips & Tricks

  • Range Slider Truc: Een <input type="range"> (de slider) retourneert standaard zijn .value als een "String" (tekst), NIET als een Number! In de code zie je const length = +lengthEl.value;. Dat rare plusje + is een Javascript trucje om een 'string' getal instantly om te toveren in een 'number' getal!

⚠️ Veelvoorkomende Fouten

Fout 1: De "Lege Vijver" Crash

// ❌ LET OP - Wat als de domme/eigenwijze gebruiker ALLE checkboxes uit vinkt, en dan op Genereer drukt? Dan is passwordList === '' (Leeg!). Dan crasht the Math.random() loop omdat hij een index probeert te zoeken in het niks! 
// OPLOSSING: Vang het af in je functie als de pool leeg is!
if (passwordList === '') {
    return 'Selecteer een optie';
}

Data Logic

Genereer Array elementen, teken ze als staafdiagram, en zie live via JavaScript hoe het Bubble Sort algoritme werkt!

📝 Stap-voor-Stap Uitleg

Stap 1: Balken (Bars) Genereren

We maken een Array van willekeurige getallen en tekenen voor elk getal een HTML <div> element. De in-line CSS height geven we de waarde van het getal in een percentage, zodat grotere getallen langere balken worden.

// Genereer random hoogte in procenten (5 tot 100)
const waarde = Math.floor(Math.random() * 95) + 5;
array.push(waarde);

// Maak de HTML div!
const balk = document.createElement('div');
balk.classList.add('bar');

// CSS Styling direct updaten met Template Literals
balk.style.height = `\${waarde}%`;
container.appendChild(balk);

Stap 2: Het "Bubble Sort" Algoritme

Bubble Sort is één van de makkelijkste manieren om the sorteren. Het Algoritme vergelijkt element 0 altijd met element 1. Is 0 groter? Wissel (Swap) ze dan van plek! Ga daarna door naar element 1 en 2. Het effect heet Bubbeling, omdat de grootste waarde telkens opschuift naar rechts (boven drijft).

if (array[j] > array[j + 1]) {
    // Wissel de waarden in the Data Array
    let tijdelijk = array[j];
    array[j] = array[j + 1];
    array[j + 1] = tijdelijk;

    // Wissel the Hoogtes ook VISUEEL aan The DOM in the UI!
    balken[j].style.height = `\${array[j]}%`;
    balken[j + 1].style.height = `\${array[j + 1]}%`;
}

Stap 3: Asynchrone Pauzes (Promise Sleep)

Een computerrekensnelheid is miljoenen the bewerkingen per seconde. Als we geen "pauze" inbouwen, is The Javascript binnen 1 milliseconde klaar met sorteren. Met async/await kunnen we het algoritme de illusie van animatie geven.

// Zelfgebouwde "Sleep" functie in JavaScript met Promises
function sleep(ms) {
    return new Promise(resolve => setTimeout(resolve, ms));
}

// Binnen the sorteer Loop... wacht voor 15 milliseconden!
await sleep(PAUSE_DURATION);

Uitleg: We veranderen tijdelijk de CSS klasse van the indexen die we "vergelijken" naar Geel. We wachten 15 ms, en als we écht the swappen, maken we ze Rood. En wachten nogmaals 15 ms!

💡 Tips & Tricks

  • Flexbox Flex-End: Waarom staan de balken netjes naar the onderkant van het the scherm the 'groeien' in the grafiek? Door de .bars-container de css list properties display: flex en align-items: flex-end te geven! Daardoor duwt flexbox alle items naar beneden, en groeien the height percentages als het ware "omhoog".

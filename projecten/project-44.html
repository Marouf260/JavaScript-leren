Project 44: Stopwatch & Laps

Data Logic

Maak een strakke, op milliseconden-nauwkeurige stopwatch met ronde-tijd (lap) functionaliteiten.

📝 Stap-voor-Stap Uitleg

Stap 1: De "Delta Tijd" Formule

Als je een klok maakt die elke seconde +1 doet, loopt deze na verloop van tijd scheef door "browser lag" (de browser is soms even druk met andere dingen). Daarom slaan we de exacte starttijd op, en trekken we die continu af van het exacte "nu". Dit heet de tijd-delta berekenen!

// Date.now() geeft de huidige tijd in een reusachtig getal milliseconden
// Bijv: 1735002000000 (Sinds 1 januari 1970)
let startTime = 0;
let elapsedTime = 0;

// Op Klik "Start":
// Waarom minus elapsedTime? Zodat als we 'Pauzeren' en weer doorgaan, de klok niet verspringt!
startTime = Date.now() - elapsedTime;

// Binnen the update loop: Actuele tijd = Nu - Starttijd
elapsedTime = Date.now() - startTime;

Stap 2: Tijd Formatteren (Wiskunde)

Dat reusachtige getal aan verstreken milliseconden (elapsedTime) moeten we begrijpelijk maken. Daarvoor gebruiken we "Modulo rekenen" (%), oftewel de restwaarde delen.

// Stap 1: Hoeveel uren / minuten / seconden passen er in het getal?
let totaalSeconden = Math.floor(elapsedTime / 1000);

// Uren, Minuten en Seconden destilleren (Modulo the the rescue)
let uren = Math.floor(totaalSeconden / 3600);
let minuten = Math.floor((totaalSeconden / 60) % 60);
let seconden = totaalSeconden % 60;

// Milliseconden (We knippen het laatste cijfer weg via delen door 10 (0 tot 99))
let mili = Math.floor((elapsedTime % 1000) / 10);

Stap 3: Cijfers opvullen met 0 (padStart)

Een klok mag nooit "1:2:5.4" laten zien. Dat moet zijn "01:02:05.04". Gelukkig heeft JavaScript hier een method voor genaamd .padStart()!

// Maak van het getal eerst een string .toString()
// .padStart(2, '0') betekent: "Zorg dat het minstens 2 karakters is, zo niet, vul op met een '0' aan de voorkant"
let mooiSeconden = seconden.toString().padStart(2, "0");

return `\${mooiMinuten}:\${mooiSeconden}.\${mooiMili}`;

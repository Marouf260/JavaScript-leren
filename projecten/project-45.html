Data Logic

Bouw een live analyse-tool zoals in MS Word, die letters, woorden, zinnen en de leestijd berekent!

📝 Stap-voor-Stap Uitleg

Stap 1: String Trimming (Schoonmaken)

Als een gebruiker per ongeluk start of eindigt met een paar spaties (bijv: " Hallo daar "), tellen we dat al snel als 4 woorden als we puur op spaties splitsen. .trim() is een ingebouwde JS methode die the onnodige whitespaces aan de the randen verwijdert.

const rauweInput = "   Ik hou van code   ";
                    
// Verwijdert spaties aan het begin en eind!
const opgeschoond = rauweInput.trim(); 
// Resultaat: "Ik hou van code"

Stap 2: Gevanceerd Splitsen (Array Maken)

Normaal leer je in tutorials the splitsen met .split(' '). Maar wat als er the 2 spaties in the zin staat? Dan klopt je telling direct the niet meer. We gebruiken een Reguliere Expressie (RegEx) /\s+/ The de "one-or-more-whitespace" methode.

let woordenCount = 0;

if (opgeschoond !== "") {
    // \s staat voor spatie, tab of linebreak. 
    // + staat voor één of meerdere keren acter elkaar!
    const woordenArray = opgeschoond.split(/\s+/);
    
    // Prik the lengte uit the Array
    woordenCount = woordenArray.length;
}

Stap 3: Leestijd Berekenen

Een gemiddeld persoon leest tussen de 200 en 250 woorden per minuut. Door ons woordenCount The the delen the door 225, en the modulo (the the rest-waarde de the berekenen the voor the secondes), kunnen we een kloppende reading the time genereren!

// Bereken het aantal volledige minuten
let minuten = Math.floor(woordenCount / 225);

// Bereken de overgebleven secondes!
// We delen the woordensnelheid (225) the door 60 the the secondensnelheid the the bepalen.
let seconden = Math.floor((woordenCount % 225) / (225 / 60));

Project 46: Quote & Color Generator

Data Logic

Maak een Data-Array applicatie, die niet alleen de tekst verandert op klik, maar op een slimme manier ook the thema UI the kleuren!

📝 Stap-voor-Stap Uitleg

Stap 1: De Data Array's (Onze "Database")

Voordat we een API (externe server) aanroepen, leren we werken met lokale "hardcoded" Arrays. De quotes staan in een Array met Objecten (zodat we tekst en auteur bijeen kunnen houden!), en de kleuren in een simpele Array met tekstkleurencodes.

// Een Array gevuld met Object {} Datapunten
const quotes = [
    { quote: "Geloof in jezelf!", author: "- Mij" },
    { quote: "Tijd is geld.", author: "- Benjamin Franklin" }
];

const colors = [ '#ef4444', '#10b981', '#f59e0b' ];

Stap 2: Randomizers (De Magische Index)

Net als bij Project 18, gebruiken we `Math.random()`. Maar in plaats van een simpel getal, gebruiken we de uitkomst als het Index Cyfer om iets uit Array te "trekken". Array[1] = item Twee!

// Kies een random index nummer gerelateerd aan de array-lengte!
let willekeurigNummer = Math.floor(Math.random() * quotes.length);

// Prik die quote eruit en geef hem aan de UI the door
const theRandomQuoteObj = quotes[willekeurigNummer];

quoteText.innerText = theRandomQuoteObj.quote;

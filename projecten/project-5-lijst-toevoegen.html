Project 5: Lijst Toevoegen

Beginner

Voeg items toe aan een lijst via een input veld.

📚 Wat ga je leren?

In dit project leer je hoe je dynamisch nieuwe elementen aan de DOM kunt toevoegen. Je leert:

  • createElement() gebruiken om nieuwe HTML elementen te maken
  • appendChild() gebruiken om elementen toe te voegen aan de DOM
  • Input waarden lezen en valideren
  • Keyboard events gebruiken (Enter toets)

🎯 Belangrijke Concepten

  • document.createElement() - Maakt een nieuw HTML element aan
  • appendChild() - Voegt een element toe als kind aan een ander element
  • input.value - Eigenschap om de waarde van een input veld te lezen
  • trim() - Methode om spaties aan begin/einde te verwijderen
  • keypress event - Event dat afgaat wanneer een toets wordt ingedrukt

📝 Stap-voor-Stap Uitleg

Stap 1: Elementen selecteren

Selecteer het input veld, de knop en de lijst:

const input = document.getElementById('list-input');
const knop = document.getElementById('add-btn');
const lijst = document.getElementById('item-list');

Uitleg: We hebben alle drie de elementen nodig: input om tekst te lezen, knop om actie te triggeren, en lijst om items aan toe te voegen.

Stap 2: Input valideren

Controleer of de input niet leeg is:

if (input.value.trim() !== '') {
    // Voeg item toe
}

Uitleg: trim() verwijdert spaties aan begin en einde. We controleren of er daadwerkelijk tekst is ingevoerd.

Stap 3: Nieuw element maken

Maak een nieuw <li> element:

const nieuwItem = document.createElement('li');
nieuwItem.textContent = input.value;

Uitleg: createElement maakt een nieuw element aan, maar voegt het nog niet toe aan de pagina. We zetten de tekst met textContent.

Stap 4: Element toevoegen aan lijst

Voeg het nieuwe element toe aan de lijst en leeg het input veld:

lijst.appendChild(nieuwItem);
input.value = ''; // Leeg input

Uitleg: appendChild voegt het element toe als laatste kind van de lijst. We legen het input veld zodat de gebruiker een nieuw item kan invoeren.

Stap 5: Enter toets ondersteuning

Voeg keyboard event listener toe voor Enter toets:

input.addEventListener('keypress', function(e) {
    if (e.key === 'Enter') {
        knop.click();
    }
});

Uitleg: Wanneer de gebruiker Enter drukt, triggeren we de klik op de knop zodat dezelfde logica wordt uitgevoerd.

💻 Complete Code

// Stap 1: Selecteer elementen
const input = document.getElementById('list-input');
const knop = document.getElementById('add-btn');
const lijst = document.getElementById('item-list');

// Stap 2-4: Event listener voor knop
knop.addEventListener('click', function() {
    if (input.value.trim() !== '') {
        const nieuwItem = document.createElement('li');
        nieuwItem.textContent = input.value;
        lijst.appendChild(nieuwItem);
        input.value = '';
    }
});

// Stap 5: Enter toets ondersteuning
input.addEventListener('keypress', function(e) {
    if (e.key === 'Enter') {
        knop.click();
    }
});

✅ Instructies:

  • Selecteer input veld, knop en lijst elementen
  • Voeg click event listener toe aan knop
  • Valideer input met trim()
  • Maak nieuw li element met createElement
  • Voeg element toe met appendChild
  • Leeg input veld na toevoegen

💡 Tips & Tricks

  • trim() is belangrijk: Zonder trim() kunnen gebruikers lege items toevoegen met alleen spaties
  • appendChild vs insertBefore: Gebruik insertBefore om items aan het begin toe te voegen
  • innerHTML alternatief: Je kunt ook lijst.innerHTML += '<li>' + input.value + '</li>' gebruiken, maar createElement is veiliger
  • Focus behouden: Gebruik input.focus() na toevoegen zodat gebruiker direct verder kan typen

⚠️ Veelvoorkomende Fouten

Fout 1: Vergeten input te valideren

// ❌ FOUT - Lege items kunnen worden toegevoegd
const nieuwItem = document.createElement('li');
nieuwItem.textContent = input.value; // Kan leeg zijn!

// ✅ GOED - Valideer eerst
if (input.value.trim() !== '') {
    const nieuwItem = document.createElement('li');
    nieuwItem.textContent = input.value;
}

Fout 2: Vergeten element toe te voegen aan DOM

// ❌ FOUT - Element bestaat maar is niet zichtbaar
const nieuwItem = document.createElement('li');
nieuwItem.textContent = input.value; // Vergeet appendChild!

// ✅ GOED - Voeg toe aan DOM
const nieuwItem = document.createElement('li');
nieuwItem.textContent = input.value;
lijst.appendChild(nieuwItem);

Fout 3: Verkeerde tag naam

// ❌ FOUT - Tag naam moet lowercase zijn
const nieuwItem = document.createElement('LI'); // Werkt maar niet standaard

// ✅ GOED - Gebruik lowercase
const nieuwItem = document.createElement('li');

🚀 Uitbreidingsmogelijkheden

Probeer deze uitdagingen om je vaardigheden te verbeteren:

  • Verwijder functionaliteit: Voeg een verwijder knop toe aan elk item
  • Edit functionaliteit: Laat gebruikers items bewerken door erop te dubbelklikken
  • Sorteren: Voeg knoppen toe om lijst alfabetisch te sorteren
  • LocalStorage: Bewaar items zodat ze behouden blijven na refresh
  • Drag & drop: Laat gebruikers items herordenen door ze te slepen

Voorbeeld: Verwijder Functionaliteit

const nieuwItem = document.createElement('li');
nieuwItem.textContent = input.value;

// Voeg verwijder knop toe
const verwijderBtn = document.createElement('button');
verwijderBtn.textContent = 'X';
verwijderBtn.addEventListener('click', function() {
    nieuwItem.remove();
});

nieuwItem.appendChild(verwijderBtn);
lijst.appendChild(nieuwItem);

Project 6: To-Do Lijst

Intermediate

Maak een to-do lijst waar je items kunt toevoegen en verwijderen.

📚 Wat ga je leren?

In dit project leer je een complete interactieve applicatie te bouwen. Je leert:

  • Complexe DOM structuren maken met meerdere elementen
  • Event listeners toevoegen aan dynamisch gemaakte elementen
  • Elementen verwijderen met remove()
  • Checkbox state gebruiken om styling te wijzigen
  • Functies organiseren voor herbruikbare code

🎯 Belangrijke Concepten

  • element.remove() - Verwijdert een element volledig uit de DOM
  • checkbox.checked - Boolean eigenschap die aangeeft of checkbox aangevinkt is
  • cssText - Eigenschap om meerdere CSS styles tegelijk in te stellen
  • Functies - Herbruikbare code blokken die je kunt aanroepen
  • Event delegation - Event listeners op parent elementen voor dynamische kinderen

Stap 2: Li element maken met styling

Maak een li element en stel styling in:

const li = document.createElement('li');
li.style.cssText = 'display: flex; align-items: center; padding: 0.5rem; margin: 0.5rem 0; background: #f0f0f0; border-radius: 4px;';

Uitleg: cssText stelt meerdere CSS eigenschappen tegelijk in. Flexbox zorgt voor horizontale layout.

Stap 3: Checkbox maken en event listener

Maak checkbox en luister naar change events:

const checkbox = document.createElement('input');
checkbox.type = 'checkbox';
checkbox.addEventListener('change', function() {
    span.style.textDecoration = this.checked ? 'line-through' : 'none';
    span.style.opacity = this.checked ? '0.5' : '1';
});

Uitleg: De ternary operator (? :) is een korte if/else. this verwijst naar het checkbox element.

Stap 4: Verwijder knop maken

Maak verwijder knop met remove() functionaliteit:

const deleteBtn = document.createElement('button');
deleteBtn.textContent = 'Verwijder';
deleteBtn.addEventListener('click', function() {
    li.remove();
});

Uitleg: remove() verwijdert het element en alle kinderen uit de DOM.

Stap 5: Elementen samenvoegen

Voeg alle elementen toe aan de li en li aan de lijst:

li.appendChild(checkbox);
li.appendChild(span);
li.appendChild(deleteBtn);
todoList.appendChild(li);

Uitleg: We bouwen de structuur op van binnen naar buiten: eerst kinderen aan li, dan li aan lijst.

💻 Complete Code

const input = document.getElementById('todo-input');
const addBtn = document.getElementById('todo-add');
const todoList = document.getElementById('todo-list');

function voegTodoToe() {
    if (input.value.trim() === '') return;
    
    const li = document.createElement('li');
    li.style.cssText = 'display: flex; align-items: center; padding: 0.5rem; margin: 0.5rem 0; background: #f0f0f0; border-radius: 4px;';
    
    const checkbox = document.createElement('input');
    checkbox.type = 'checkbox';
    checkbox.style.marginRight = '0.5rem';
    
    const span = document.createElement('span');
    span.textContent = input.value;
    span.style.flex = '1';
    
    const deleteBtn = document.createElement('button');
    deleteBtn.textContent = 'Verwijder';
    deleteBtn.style.cssText = 'background: #e74c3c; color: white; border: none; padding: 0.25rem 0.5rem; border-radius: 4px; cursor: pointer;';
    
    checkbox.addEventListener('change', function() {
        span.style.textDecoration = this.checked ? 'line-through' : 'none';
        span.style.opacity = this.checked ? '0.5' : '1';
    });
    
    deleteBtn.addEventListener('click', function() {
        li.remove();
    });
    
    li.appendChild(checkbox);
    li.appendChild(span);
    li.appendChild(deleteBtn);
    todoList.appendChild(li);
    
    input.value = '';
}

addBtn.addEventListener('click', voegTodoToe);
input.addEventListener('keypress', function(e) {
    if (e.key === 'Enter') voegTodoToe();
});

💡 Tips & Tricks

  • Functies: Gebruik functies om code herbruikbaar te maken en te organiseren
  • Ternary operator: condition ? value1 : value2 is korter dan if/else voor simpele waarden
  • this keyword: In event listeners verwijst this naar het element dat het event heeft getriggerd
  • remove() vs removeChild(): remove() is moderner en eenvoudiger dan parent.removeChild(child)

⚠️ Veelvoorkomende Fouten

Fout 1: Vergeten elementen toe te voegen aan DOM

// ✅ GOED - Zet type expliciet
const checkbox = document.createElement('input');
checkbox.type = 'checkbox';

Voorbeeld: LocalStorage

// Bewaar todos
function bewaarTodos() {
    const todos = Array.from(todoList.children).map(li => ({
        tekst: li.querySelector('span').textContent,
        completed: li.querySelector('input').checked
    }));
    localStorage.setItem('todos', JSON.stringify(todos));
}

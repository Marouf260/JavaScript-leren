Project 7: Kleuren Picker

Intermediate

Maak een color picker die de achtergrondkleur wijzigt.

📚 Wat ga je leren?

In dit project leer je werken met HTML5 input types en real-time updates. Je leert:

  • De HTML5 color input type gebruiken
  • Input events gebruiken voor real-time updates
  • this.value gebruiken om input waarden te lezen
  • Dynamisch tekst en styling updaten

🎯 Belangrijke Concepten

  • <input type="color"> - HTML5 color picker input type
  • input event - Event dat afgaat bij elke wijziging (niet alleen bij blur)
  • this.value - Eigenschap om de huidige waarde van een input te lezen
  • Real-time updates - Directe feedback zonder form submit
  • Hex kleurcodes - Kleuren in #RRGGBB formaat

📝 Stap-voor-Stap Uitleg

Stap 1: Elementen selecteren

Selecteer het preview element en de color picker:

const preview = document.getElementById('color-preview');
const picker = document.getElementById('color-picker');

Uitleg: We hebben beide elementen nodig: preview om kleur te tonen en picker om kleur te kiezen.

Stap 2: Input event listener

Luister naar input events (niet change, want input geeft real-time updates):

picker.addEventListener('input', function() {
    preview.style.backgroundColor = this.value;
    preview.textContent = 'Gekozen kleur: ' + this.value;
});

Uitleg: input event gaat af bij elke wijziging, terwijl change alleen afgaat wanneer het veld focus verliest. this.value geeft de gekozen hex kleurcode.

💻 Complete Code

// Stap 1: Selecteer elementen
const preview = document.getElementById('color-preview');
const picker = document.getElementById('color-picker');

// Stap 2: Input event listener
picker.addEventListener('input', function() {
    preview.style.backgroundColor = this.value;
    preview.textContent = 'Gekozen kleur: ' + this.value;
});

✅ Instructies:

  • Maak een <input type="color"> element in HTML
  • Selecteer preview element en color picker
  • Voeg input event listener toe (niet change!)
  • Update backgroundColor en textContent met this.value

💡 Tips & Tricks

  • input vs change: Gebruik input voor real-time updates, change alleen bij blur
  • this.value: In event listeners verwijst this naar het input element
  • value attribuut: Je kunt een standaard kleur instellen met value="#3498db"
  • Kleur conversie: Je kunt hex naar RGB converteren met JavaScript functies

⚠️ Veelvoorkomende Fouten

Fout 1: change event gebruiken in plaats van input

// ❌ FOUT - Update alleen bij blur, niet real-time
picker.addEventListener('change', function() {
    preview.style.backgroundColor = this.value;
});

// ✅ GOED - Real-time updates met input event
picker.addEventListener('input', function() {
    preview.style.backgroundColor = this.value;
});

Fout 2: Verkeerde input type

// ❌ FOUT - Geen color picker
<input type="text" id="color-picker">

// ✅ GOED - Gebruik color type
<input type="color" id="color-picker">

Fout 3: Vergeten this te gebruiken

// ❌ FOUT - picker.value werkt maar this is beter
picker.addEventListener('input', function() {
    preview.style.backgroundColor = picker.value; // Werkt maar niet flexibel
});

// ✅ GOED - Gebruik this voor flexibiliteit
picker.addEventListener('input', function() {
    preview.style.backgroundColor = this.value;
});

🚀 Uitbreidingsmogelijkheden

Probeer deze uitdagingen om je vaardigheden te verbeteren:

  • RGB sliders: Maak aparte sliders voor R, G, B waarden
  • Kleur geschiedenis: Bewaar laatst gebruikte kleuren
  • Kleur palet: Toon vooraf gedefinieerde kleuren
  • Gradient generator: Genereer CSS gradients met twee kleuren
  • Kleur contrast checker: Controleer of tekst leesbaar is op achtergrond

Voorbeeld: RGB Sliders

// HTML: <input type="range" id="r-slider" min="0" max="255">
const rSlider = document.getElementById('r-slider');
const gSlider = document.getElementById('g-slider');
const bSlider = document.getElementById('b-slider');

function updateKleur() {
    const r = rSlider.value;
    const g = gSlider.value;
    const b = bSlider.value;
    preview.style.backgroundColor = `rgb(${r}, ${g}, ${b})`;
}

[rSlider, gSlider, bSlider].forEach(slider => {
    slider.addEventListener('input', updateKleur);
});

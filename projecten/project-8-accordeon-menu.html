Project 8: Accordeon Menu

Intermediate

Maak een accordeon menu dat open en dicht klapt.

📚 Wat ga je leren?

In dit project leer je werken met meerdere elementen en DOM navigatie. Je leert:

  • querySelectorAll gebruiken voor meerdere elementen
  • nextElementSibling gebruiken om naar volgende element te navigeren
  • forEach gebruiken om over arrays/NodeLists te itereren
  • State management met meerdere elementen

🎯 Belangrijke Concepten

  • querySelectorAll() - Selecteert alle elementen die matchen met een selector
  • nextElementSibling - Eigenschap om naar het volgende sibling element te gaan
  • forEach() - Methode om over een array/NodeList te itereren
  • NodeList - Lijst van DOM elementen (lijkt op array maar is het niet)
  • Exclusive toggle - Alleen één item open tegelijk

📝 Stap-voor-Stap Uitleg

Stap 1: Alle headers selecteren

Selecteer alle accordeon headers met querySelectorAll:

const headers = document.querySelectorAll('.accordion-header');

Uitleg: querySelectorAll retourneert een NodeList met alle elementen die de class hebben. We gebruiken een class selector (begint met .).

Stap 2: Event listener toevoegen aan elke header

Gebruik forEach om over alle headers te itereren:

headers.forEach(header => {
    header.addEventListener('click', function() {
        // Logica
    });
});

Uitleg: forEach voert een functie uit voor elk element. Arrow function (=>) is een korte functie syntax.

Stap 3: Content element vinden met nextElementSibling

Gebruik nextElementSibling om het content element te vinden:

const content = this.nextElementSibling;
const isOpen = content.style.display === 'block';

Uitleg: nextElementSibling geeft het volgende element op hetzelfde niveau in de DOM tree. We controleren of het al open is.

Stap 4: Sluit alle andere secties

Sluit eerst alle content elementen:

document.querySelectorAll('.accordion-content').forEach(item => {
    item.style.display = 'none';
});

Uitleg: We sluiten eerst alles, dan openen we alleen de geklikte sectie. Dit zorgt voor exclusive behavior (slechts één open).

Stap 5: Open geklikte sectie als het gesloten was

Open alleen als het nog niet open was:

if (!isOpen) {
    content.style.display = 'block';
}

Uitleg: Als de sectie gesloten was (!isOpen), openen we hem. Als hij al open was, blijft hij gesloten (toggle effect).

💻 Complete Code

const headers = document.querySelectorAll('.accordion-header');

headers.forEach(header => {
    header.addEventListener('click', function() {
        const content = this.nextElementSibling;
        const isOpen = content.style.display === 'block';
        
        // Sluit alle andere
        document.querySelectorAll('.accordion-content').forEach(item => {
            item.style.display = 'none';
        });
        
        // Open deze als het gesloten was
        if (!isOpen) {
            content.style.display = 'block';
        }
    });
});

✅ Instructies:

  • Selecteer alle headers met querySelectorAll
  • Gebruik forEach om event listeners toe te voegen
  • Gebruik nextElementSibling om content te vinden
  • Sluit eerst alle content elementen
  • Open alleen geklikte sectie als die gesloten was

💡 Tips & Tricks

  • querySelectorAll vs getElementsByClassName: querySelectorAll is moderner en flexibeler
  • nextElementSibling vs nextSibling: nextElementSibling geeft alleen elementen, nextSibling geeft ook tekst nodes
  • Arrow functions: header => {} is hetzelfde als function(header) {}
  • Exclusive vs multiple: Voor meerdere open tegelijk, verwijder de "sluit alle andere" logica

⚠️ Veelvoorkomende Fouten

Fout 1: Vergeten forEach te gebruiken

// ❌ FOUT - Voegt listener alleen toe aan eerste element
const headers = document.querySelectorAll('.accordion-header');
headers.addEventListener('click', ...); // Werkt niet!

// ✅ GOED - Gebruik forEach
headers.forEach(header => {
    header.addEventListener('click', ...);
});

Fout 2: Verkeerde sibling property

// ❌ FOUT - nextSibling kan tekst node zijn
const content = this.nextSibling; // Kan whitespace zijn!

// ✅ GOED - nextElementSibling geeft alleen elementen
const content = this.nextElementSibling;

Fout 3: Vergeten alle andere te sluiten

// ❌ FOUT - Meerdere secties kunnen open zijn
content.style.display = 'block'; // Vergeet andere te sluiten

// ✅ GOED - Sluit eerst alles
document.querySelectorAll('.accordion-content').forEach(item => {
    item.style.display = 'none';
});
if (!isOpen) content.style.display = 'block';

🚀 Uitbreidingsmogelijkheden

Probeer deze uitdagingen om je vaardigheden te verbeteren:

  • Animatie: Voeg slide down/up animatie toe met CSS transitions
  • Icons: Voeg + en - icons toe die veranderen bij open/sluiten
  • Meerdere open: Laat meerdere secties tegelijk open zijn
  • Keyboard navigatie: Gebruik pijltjestoetsen om tussen secties te navigeren
  • LocalStorage: Bewaar welke secties open waren

Voorbeeld: Meerdere Open Tegelijk

headers.forEach(header => {
    header.addEventListener('click', function() {
        const content = this.nextElementSibling;
        const isOpen = content.style.display === 'block';
        
        // Toggle alleen deze sectie
        content.style.display = isOpen ? 'none' : 'block';
    });
});

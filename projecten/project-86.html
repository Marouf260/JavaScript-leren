📝 Stap-voor-Stap Uitleg

Stap 1: performance.getEntriesByType('navigation')

De browser houdt bij hoe lang elke stap van het laden van de pagina duurde. We kunnen dit opvragen via de Performance API!

const nav = performance.getEntriesByType('navigation')[0];
const domReady = Math.round(nav.domContentLoadedEventEnd); // in ms
const fullLoad = Math.round(nav.loadEventEnd); // in ms

Stap 2: performance.now() voor benchmarks

Met performance.now() kun je super nauwkeurig meten hoe lang jouw eigen code duurt om te draaien. Dit werkt met microseconden precisie!

const start = performance.now();
// ... jouw te meten code ...
const eind = performance.now();
console.log(`Duurde ${eind - start} ms`);

💡 Tips & Tricks

  • setTimeout trick: Om de UI te laten updaten vóórdat de zware for-loop de thread blokkeert, wrappen we de code in een setTimeout(() => {...}, 50). Dit geeft de browser tijd om de knop-animatie te renderen.

📝 Stap-voor-Stap Uitleg

Stap 1: Het visibilitychange Event

De Page Visibility API laat ons weten wanneer de gebruiker van tabblad wisselt of de browser minimaliseert. We luisteren naar het visibilitychange event op het document. 

document.addEventListener('visibilitychange', () => {
    if (document.hidden) {
        // Gebruiker is WEG - pauzeer alles!
        video.pause();
    } else {
        // Gebruiker is TERUG - hervat!
        video.play();
    }
});

Stap 2: document.hidden

De boolean document.hidden is true als de pagina niet zichtbaar is. Gebruik dit om resource-intensieve taken te pauzeren als de gebruiker niet kijkt - zo bespaar je batterij en CPU!

✅ Jouw Opdracht:

  • Klik Play op de video!
  • Wissel van tabblad - de video pauzeert automatisch!
  • Kom terug - de video hervat!

💡 Tips & Tricks

  • Gebruik cases: Pauzeer animaties, stop muziek, pauzeer timers, stop API-polls als de gebruiker niet naar de pagina kijkt. Dit bespaart enorm veel resources!

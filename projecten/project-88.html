📝 Stap-voor-Stap Uitleg

Stap 1: De Database Openen

IndexedDB is een echte database in de browser. Het openen werkt callback-gebaseerd. De onupgradeneeded event is de plek om je "tabellen" (objectStores) aan te maken.

const req = window.indexedDB.open('MijnDb', 1);
req.onupgradeneeded = (e) => {
    e.target.result.createObjectStore('notities',
        { keyPath: 'id', autoIncrement: true });
};
req.onsuccess = (e) => db = e.target.result;

Stap 2: Data Lezen & Schrijven via Transacties

Elke operatie (lezen/schrijven) gaat via een Transactie. Schrijven = 'readwrite', Lezen = 'readonly'.

// Schrijven
const tx = db.transaction(['notities'], 'readwrite');
tx.objectStore('notities').add({ tekst: 'Hallo!' });

// Lezen
tx.objectStore('notities').getAll().onsuccess = (e) => 
    console.log(e.target.result);

💡 Tips & Tricks

  • vs localStorage: localStorage slaat alleen strings op. IndexedDB slaat volledige JavaScript Objecten op, is asynchroon en heeft vrijwel geen opslaglimiet!

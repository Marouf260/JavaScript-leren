Clipboard API

Advanced APIs

📝 Stap-voor-Stap Uitleg

Stap 1: Schrijven naar het Klembord

De Clipboard API laat ons programmatisch tekst kopiëren naar het klembord. Dit is de moderne manier in plaats van de oude document.execCommand('copy')!

await navigator.clipboard.writeText('Tekst om te kopiëren!');
console.log('Gekopieerd!'); // Werkt alleen na await!

Stap 2: Lezen van het Klembord

Lezen van het klembord (readText()) vereist toestemming van de gebruiker! De browser zal een popup tonen. Dit is een veiligheidsmaatregel.

const tekst = await navigator.clipboard.readText();
textarea.value = tekst; // Zet geplakte tekst in textarea

✅ Jouw Opdracht:

  • Klik "Kopiëren" om de tekst naar je klembord te kopiëren!
  • Klik daarna "Plak Programmatisch" om het te plakken!
  • Probeer ook Ctrl+V in het tekstvak hieronder.

💡 Tips & Tricks

  • Altijd await: Beide clipboard-functies zijn asynchronous! Vergeet het await niet, anders heb je een Promise in plaats van de echte tekst.
  • HTTPS Required: Net als de andere browser-APIs werkt de Clipboard API alleen op HTTPS of localhost.

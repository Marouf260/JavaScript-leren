Project 9: Tab Interface

Intermediate

Maak een tab interface waar je tussen verschillende secties kunt wisselen.

📚 Wat ga je leren?

In dit project leer je werken met data attributen en complexe state management. Je leert:

  • data-* attributen gebruiken om elementen te koppelen
  • classList gebruiken om classes toe te voegen/verwijderen
  • Template literals gebruiken voor dynamische selectors
  • State synchroniseren tussen meerdere elementen

🎯 Belangrijke Concepten

  • data-* attributen - HTML attributen om custom data op te slaan
  • dataset - Eigenschap om data-* attributen te lezen (dataset.tab)
  • classList - Object met methodes om classes te beheren (add, remove, toggle)
  • Template literals - Strings met ${variabele} voor dynamische waarden
  • Attribute selectors - CSS selectors zoals [data-content="1"]

📝 Stap-voor-Stap Uitleg

Stap 1: Elementen selecteren

Selecteer alle tab knoppen en content secties:

const tabBtns = document.querySelectorAll('.tab-btn');
const tabContents = document.querySelectorAll('.tab-content');

Uitleg: We hebben beide NodeLists nodig om state te synchroniseren tussen knoppen en content.

Stap 2: Event listener met data attribuut lezen

Lees het data-tab attribuut om te weten welke tab geklikt is:

tabBtns.forEach(btn => {
    btn.addEventListener('click', function() {
        const tabId = this.dataset.tab;
        // tabId bevat de waarde van data-tab
    });
});

Uitleg: dataset.tab leest het data-tab attribuut. Dit koppelt knoppen aan content.

Stap 3: Verwijder active van alle elementen

Reset alle tabs en content naar inactive state:

tabBtns.forEach(b => {
    b.classList.remove('active');
    b.style.background = '#ccc';
});
tabContents.forEach(c => {
    c.classList.remove('active');
    c.style.display = 'none';
});

Uitleg: We resetten eerst alles, dan activeren we alleen de geselecteerde tab. Dit voorkomt meerdere actieve tabs.

Stap 4: Active state toevoegen aan geklikte tab

Gebruik template literal om juiste content te vinden:

this.classList.add('active');
this.style.background = '#667eea';
const content = document.querySelector(`[data-content="${tabId}"]`);
content.classList.add('active');
content.style.display = 'block';

Uitleg: Template literal (`...`) laat variabelen toe in strings. [data-content="${tabId}"] zoekt element met matching data-content.

💻 Complete Code

const tabBtns = document.querySelectorAll('.tab-btn');
const tabContents = document.querySelectorAll('.tab-content');

tabBtns.forEach(btn => {
    btn.addEventListener('click', function() {
        const tabId = this.dataset.tab;
        
        // Verwijder active van alle tabs
        tabBtns.forEach(b => {
            b.classList.remove('active');
            b.style.background = '#ccc';
            b.style.color = '#333';
        });
        tabContents.forEach(c => {
            c.classList.remove('active');
            c.style.display = 'none';
        });
        
        // Voeg active toe aan geklikte tab
        this.classList.add('active');
        this.style.background = '#667eea';
        this.style.color = 'white';
        const content = document.querySelector(`[data-content="${tabId}"]`);
        content.classList.add('active');
        content.style.display = 'block';
    });
});

✅ Instructies:

  • Selecteer alle tab knoppen en content secties
  • Gebruik data-tab en data-content attributen voor koppeling
  • Lees tabId met dataset.tab
  • Reset eerst alle active states
  • Activeer geklikte tab en bijbehorende content

💡 Tips & Tricks

  • data-* attributen: Gebruik data-* voor custom data, niet voor styling
  • classList vs className: classList is beter voor meerdere classes, className overschrijft alles
  • Template literals: Gebruik backticks (`) voor strings met variabelen
  • Attribute selectors: [data-content="1"] selecteert elementen met specifieke data waarde

⚠️ Veelvoorkomende Fouten

Fout 1: Verkeerde data attribuut naam

// ❌ FOUT - dataset.tabId bestaat niet
const tabId = this.dataset.tabId; // data-tab-id wordt tabId

// ✅ GOED - data-tab wordt dataset.tab
const tabId = this.dataset.tab;

Fout 2: Vergeten alle andere te resetten

// ❌ FOUT - Meerdere tabs kunnen actief zijn
this.classList.add('active'); // Vergeet andere te verwijderen

// ✅ GOED - Reset eerst alles
tabBtns.forEach(b => b.classList.remove('active'));
this.classList.add('active');

Fout 3: Verkeerde template literal syntax

// ❌ FOUT - Gewone string, variabele wordt niet geïnterpreteerd
const content = document.querySelector('[data-content="tabId"]');

// ✅ GOED - Template literal met backticks
const content = document.querySelector(`[data-content="${tabId}"]`);

🚀 Uitbreidingsmogelijkheden

Probeer deze uitdagingen om je vaardigheden te verbeteren:

  • Keyboard navigatie: Gebruik pijltjestoetsen om tussen tabs te navigeren
  • Animatie: Voeg fade of slide animaties toe bij tab switch
  • URL hash: Update URL hash (#tab1) en laad juiste tab bij page load
  • Lazy loading: Laad tab content alleen wanneer tab wordt geopend
  • Close functionaliteit: Voeg X knop toe om tabs te sluiten

Voorbeeld: Keyboard Navigatie

document.addEventListener('keydown', function(e) {
    const activeTab = document.querySelector('.tab-btn.active');
    const currentIndex = Array.from(tabBtns).indexOf(activeTab);
    
    if (e.key === 'ArrowRight' && currentIndex < tabBtns.length - 1) {
        tabBtns[currentIndex + 1].click();
    } else if (e.key === 'ArrowLeft' && currentIndex > 0) {
        tabBtns[currentIndex - 1].click();
    }
});

📝 Stap-voor-Stap Uitleg

Stap 1: Vibration API

De Vibration API laat een telefoon trillen via navigator.vibrate(). Je kunt een getal meegeven (ms) of een array voor patronen!

// Simpel getal = 1 trilling van 200ms
navigator.vibrate(200);

// Array = patroon: [tril, rust, tril, rust, ...]
navigator.vibrate([100, 100, 300]); // kort, pauze, lang

Stap 2: Battery API

De Battery Status API geeft toegang tot de batterijstatus van het apparaat. Let op: Dit is deprecated op Firefox maar werkt nog in Chrome en Edge!

const battery = await navigator.getBattery();
console.log(battery.level);    // 0 tot 1 (bijv: 0.85 = 85%)
console.log(battery.charging); // true of false

✅ Jouw Opdracht:

  • Open deze pagina op je mobiel voor de trillingen!
  • Klik de knoppen om de SOS en hartslag patronen te voelen!
  • Bekijk je batterijpercentage rechts in beeld.

💡 Tips & Tricks

  • Vibrate op Desktop: Op de PC doet navigator.vibrate() niets - er is immers geen trilmotor! Test het op je telefoon voor het echte effect.
  • Battery deprecated: De Battery API werkt niet meer in Firefox en iOS Safari vanwege privacyzorgen. Chrome ondersteunt het nog wel.

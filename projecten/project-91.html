📝 Stap-voor-Stap Uitleg

Stap 1: Tekst in Spans Splitsen

De tekst wordt in losse woord-spans opgesplitst zodat elk woord individueel kan worden gecheckt en opgemaakt met CSS classes.

const woorden = zin.split(' ');
woorden.forEach(woord => {
    const span = document.createElement('span');
    span.innerText = woord + ' ';
    container.appendChild(span);
});

Stap 2: WPM Berekenen

WPM = Words Per Minute. Formule: correcte woorden gedeeld door verstreken minuten. We berekenen dit live elke seconde!

const minuten = verstrekenSeconden / 60;
const wpm = Math.round(correcteWoorden / minuten);

✅ Jouw Opdracht:

  • Klik in het tekstveld en begin met typen!
  • Typ elk woord gevolgd door een Spatie - het woord wordt dan verwerkt.
  • Probeer 60+ WPM te halen!

💡 Tips & Tricks

  • CSS Feedback: Groene achtergrond = correct, Rood = fout. Dit werkt via het simpelweg toevoegen van CSS classes aan de spans!
  • Gemiddeld: Een gemiddelde typist haalt 40 WPM. 60+ is goed, 80+ is uitstekend, 100+ is expert-niveau!

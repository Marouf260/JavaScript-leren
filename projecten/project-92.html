📝 Stap-voor-Stap Uitleg

Stap 1: CSS 3D Kaart Flip

De vloeiende kaartomkeer werkt puur via CSS! We gebruiken transform-style: preserve-3d en rotateY(180deg). JavaScript voegt alleen de class toe.

/* CSS */
.kaart-inner { transform-style: preserve-3d; transition: 0.6s; }
.kaart.geflippt .kaart-inner { transform: rotateY(180deg); }
.voorkant { backface-visibility: hidden; transform: rotateY(180deg); }
.achterkant { backface-visibility: hidden; }

Stap 2: Match Logica

We slaan de eerste kaart op als referentie. Bij de 2e klik vergelijken we de data-emoji attributen. Een match = groene glow + pointer-events: none. Geen match = na 1 seconde terugdraaien.

✅ Jouw Opdracht:

  • Klik op een kaart om hem om te draaien!
  • Vind alle 8 matching emoji-paren!
  • Probeer het te doen met zo min mogelijk missers!

💡 Tips & Tricks

  • dataset: Via element.dataset.emoji lees je het data-emoji="..." HTML-attribuut uit. Dit is de standaard manier om data bij DOM-elementen op te slaan!
  • Block Guard: De isBlokkeerd variabele voorkomt dat je een 3e kaart kunt klikken terwijl de animatie nog loopt!

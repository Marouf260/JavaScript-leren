📝 Stap-voor-Stap Uitleg

Stap 1: De Canvas API Context

Tekenen op een <canvas> gaat via de 2D Drawing Context. Dit is de "pen" waarmee je alles tekent. Je bereikt hem via canvas.getContext('2d').

const canvas = document.getElementById('mijnCanvas');
const ctx = canvas.getContext('2d');
ctx.strokeStyle = '#3b82f6'; // Kleur
ctx.lineWidth = 5;          // Dikte
ctx.beginPath();
ctx.moveTo(10, 10);   // Startpunt
ctx.lineTo(200, 100); // Eindpunt
ctx.stroke();           // Teken!

Stap 2: Responsieve Coordinaten

Het canvas heeft een vaste resolutie (800x500px) maar wordt via CSS kleiner getoond. We moeten de muiscoordinaten schalen naar de echte canvas-pixels!

const rect = canvas.getBoundingClientRect();
const scaleX = canvas.width / rect.width;

✅ Jouw Opdracht:

  • Kies een kleur en dikte, klik en sleep om te tekenen!
  • Probeer de Gum - die tekent gewoon wit over de bestaande tekening!
  • Werkt ook op touch-apparaten (mobiel/tablet)!

💡 Tips & Tricks

  • lineCap: 'round': Zorgt voor mooie ronde lijneindpunten. Zonder dit krijg je harde vierkante eindes!
  • Gum Truc: De gum is simpelweg de pen met strokeStyle = 'white'. Geen speciale API nodig!

🚀 Uitbreidingsmogelijkheden

Klaar voor MEER!? Probeer dit toe te voegen:

  • Opslaan als PNG: Gebruik canvas.toDataURL() om een download-link te maken voor de tekening!
  • Vormen: Voeg knoppen toe om rechthoeken (ctx.fillRect()) en cirkels (ctx.arc()) te tekenen!

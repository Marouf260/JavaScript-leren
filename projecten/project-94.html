📝 Stap-voor-Stap Uitleg

Stap 1: AudioContext en OscillatorNode

De Web Audio API werkt met een signaalnetwerk. Een OscillatorNode genereert de geluidsgolf op een bepaalde frequentie (Hz). We verbinden hem via een GainNode (volume) aan de speakers.

const ctx = new AudioContext();
const osc = ctx.createOscillator();
const gain = ctx.createGain();

osc.connect(gain);
gain.connect(ctx.destination); // speakers
osc.frequency.value = 440; // Hz = La (A4)
osc.type = 'triangle'; // golfvorm
osc.start();

Stap 2: Envelope (Attack & Release)

Een echte piano klinkt niet abrupt. We gebruiken exponentialRampToValueAtTime() voor een natuurlijk klingende fade-out (de envelope).

💡 Tips & Tricks

  • AudioContext Suspend: Browsers blokkeren audio totdat de gebruiker iets klikt (autoplay policy). We initialiseren de AudioContext pas bij de eerste toetsaanslag!
  • Frequenties: Muzieknoten hebben vaste frequenties: La (A4) = 440 Hz. Elke octaaf hoger verdubbelt de frequentie!

Grandmaster Project (Canvas API)

📝 Stap-voor-Stap Uitleg

Stap 1: De Game Loop

Een canvas-game werkt met een Game Loop. Elke frame (~60x per seconde) wist je het canvas en teken je alles opnieuw op de nieuwe positie. Dit geeft de illusie van beweging!

function update() {
    ctx.clearRect(0, 0, canvas.width, canvas.height);
    moveBall();
    drawBall();
    drawPaddle();
    drawBricks();
    requestAnimationFrame(update); // ~60 fps
}

Stap 2: Botsingsdetectie (Collision Detection)

We controleren elke frame of de bal de paddle, een steen of een muur raakt. Bij een treffer keren we de Y-richting om (ball.dy *= -1).

✅ Jouw Opdracht:

  • Klik Start Game of druk op S!
  • Gebruik ← / → of A / D om de plank te bewegen!
  • Breek alle stenen en verlies geen 3 levens!

💡 Tips & Tricks

  • requestAnimationFrame: Dit is efficiënter dan setInterval voor animaties. De browser synchroniseert het met de schermverversing (60Hz) voor vloeiende beweging!
  • Spin effect: We berekenen waar de bal de paddle raakt en passen ball.dx aan - zo kan je richting geven!

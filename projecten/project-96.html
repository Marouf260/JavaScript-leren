Stap 1: De iFrame als Sandbox

We gebruiken een <iframe> als zandbak (sandbox) voor de preview. We genereren dynamisch een volledige HTML-pagina als string en schrijven die met document.write(). 

const fullDoc = `<!DOCTYPE html>
<html><head><style>${css}</style></head>
<body>${html}<script>${js}</script></body>
</html>`;

const doc = iframe.contentDocument;
doc.open();
doc.write(fullDoc);
doc.close();

✅ Jouw Opdracht

  • Typ HTML, CSS en JavaScript in de editors en zie de preview live updaten!
  • Maak een animatie met CSS @keyframes of een knop die een alert toont!

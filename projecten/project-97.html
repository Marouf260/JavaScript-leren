Stap 1: Markdown Parsen met Regex

We bouwen een eigen Markdown parser met reguliere expressies (Regex). Elk Markdown-patroon wordt naar HTML vervangen met str.replace().

// ** tekst ** wordt dikgedrukt
html = html.replace(/\*\*(.*?)\*\*/g, '<strong>$1</strong>');

// # Koptekst wordt h1
html = html.replace(/^# (.*$)/gim, '<h1>$1</h1>');

✅ Jouw Opdracht

  • Typ Markdown in het linkerpaneel en zie de live HTML-preview rechts!
  • Probeer koppen (# t/m ######), **dikgedrukt**, *cursief* en lijsten!

🚀 Uitbreidingsmogelijkheden

  • Tabel Support: Voeg Regex toe voor Markdown-tabellen (| Col1 | Col2 |)!
  • Export als HTML: Voeg een download-knop toe die de gegenereerde HTML downloadt als bestand!

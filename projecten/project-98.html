Flappy Cube

Grandmaster Project (Canvas Game)

📝 Stap-voor-Stap Uitleg

Stap 1: Zwaartekracht Simuleren

In dit spel simuleren we zwaartekracht! Elke frame voegen we een kleine GRAVITY waarde toe aan de Y-snelheid. Bij een sprong zetten we de snelheid naar een negatief getal (omhoog).

const GRAVITY = 0.25;
const JUMP = -5.5; // Negatief = omhoog!

// Elke frame:
cube.velocity += GRAVITY; // Zwaartekracht
cube.y += cube.velocity;

// Bij klik:
cube.velocity = JUMP;

Stap 2: Pijpen Aanmaken & Verwijderen

Elke N frames voegen we een nieuwe pipe toe aan een array. Als een pipe het scherm uit beweegt, verwijderen we hem uit de array. Dit heet een object pool patroon.

✅ Jouw Opdracht:

  • Druk Start Game en gebruik Spatie of klik om te vliegen!
  • Vlieg door de groene pijpen zonder ze te raken!
  • Je highscore wordt opgeslagen - probeer je eigen record te breken!

💡 Tips & Tricks

  • Highscore in localStorage: De beste score wordt opgeslagen met localStorage.setItem() en wordt bij elke start ingeladen - ook na een refresh!
  • Object methoden: In dit project gebruiken we cube.draw() en pipes.update() - objecten als mini-klassen!

🚀 Uitbreidingsmogelijkheden

Klaar voor MEER!? Probeer dit toe te voegen:

  • Moeilijkheidsgraad: Verhoog de pijpensnelheid (dx) naarmate de score stijgt!
  • Geluid: Voeg punt-geluiden toe via de Web Audio API bij elke doorvlogen pijp!

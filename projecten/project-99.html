Custom Video Player

Grandmaster Project (Media API)

📝 Stap-voor-Stap Uitleg

Stap 1: HTML Video Element Aansturen

We gebruiken het native <video> element en sturen het aan via JavaScript. Het video element heeft ingebouwde methoden en events.

// Afspelen/Pauzeren
video.play();
video.pause();

// Luister naar voortgang
video.addEventListener('timeupdate', () => {
    const pct = (video.currentTime / video.duration) * 100;
    progressBar.style.width = pct + '%';
});

Stap 2: Picture in Picture & Fullscreen

Moderne browser APIs: video.requestPictureInPicture() voor PiP en element.requestFullscreen() voor volledig scherm. Dit zijn async functies!
